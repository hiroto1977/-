# みんなのデスクトップ

誰でも使えるシンプルで安全な業務用デスクトップシステム。
個人事業主から大企業まで、PC初心者から熟練ユーザーまで、幅広く利用できます。

![アプリ画面](https://via.placeholder.com/800x500/1f6feb/ffffff?text=Minna+Desktop)

## 特徴

- **完全オフライン動作** — データは端末内にだけ保存。インターネット送信なし。
- **ビルド不要** — 純粋な HTML / CSS / JavaScript。誰でも保守・改造できます。
- **PWA 対応** — ブラウザから「ホームに追加」でアプリのように使えます。
- **アクセシビリティ最優先** — 高コントラストテーマ、特大文字、キーボード操作、フォーカス可視化、ARIA 属性、スクリーンリーダー対応。
- **多様な利用者向け** — ライト / ダーク / 高コントラスト切替、4段階の文字サイズ、表示密度切替。

## 起動方法

### 1. ローカルで試す（最も簡単）

```bash
cd desktop
python3 -m http.server 8000
```

ブラウザで <http://127.0.0.1:8000/> を開きます。

### 2. 本番配置

`desktop/` 以下のファイルを任意の静的ホスティング（Nginx / Apache / S3 / GitHub Pages / Cloudflare Pages 等）に配置するだけです。サーバーサイドの処理は一切不要です。

## キーボードショートカット

| 操作 | キー |
|---|---|
| ホームへ戻る | `Esc` |
| ヘルプを開く | `F1` |
| 設定を開く | `Ctrl` + `,` |
| 新規追加（各アプリ） | `Ctrl` + `N` |

## データの保存場所

すべてのデータはブラウザの `localStorage` に保存されます。

- 容量目安: 約 5〜10MB（ブラウザによる）
- バックアップ: 設定 → 「バックアップを書き出す」で JSON 形式エクスポート
- 復元: 設定 → 「バックアップを読み込む」

## ファイル構成

```
desktop/
├── index.html            # エントリポイント
├── styles.css            # 全スタイル（テーマ・密度・文字サイズ対応）
├── app.js                # アプリ本体（ESモジュール、フレームワーク非依存）
├── manifest.webmanifest  # PWA マニフェスト
└── sw.js                 # Service Worker（オフライン対応）
```
