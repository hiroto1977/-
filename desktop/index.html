    使い方

    1. アプリを選ぶ — ホーム画面の大きなボタンから、使いたいアプリを選びます。
    2. 入力する — 各アプリで内容を入力してください。書いたものは自動で保存されます。
    3. 戻る — 左上の「戻る」または Esc でホームに戻ります。
    4. バックアップ — 設定から「バックアップを書き出す」で全データをファイル保存できます。

    よくある質問

    データはどこに保存されますか？

    このパソコン（ブラウザ）の中にだけ保存されます。インターネットには送信されません。

    別のパソコンに移したい

    「設定 → バックアップを書き出す」でファイルを保存し、移し先で「読み込む」を押してください。

    間違って消した

    各アプリの「ゴミ箱」から30日以内なら戻せます。バックアップからの復元も可能です。
