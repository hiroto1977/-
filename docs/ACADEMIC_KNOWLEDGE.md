# 学術知識 確証済みベース（ACADEMIC_KNOWLEDGE）

Service Hub が参照する学際的知識（経済学・経営学・人間科学・ビジネス法務・情報社会学および関連分野）の
基礎概念・理論は、`src/renderer/data/academicKnowledge.ts` の `VERIFIED_CONCEPTS` に採録する。

> ⚠️ 本データは一般的な学術知識の要約であり、特定の意思決定への助言ではありません。実務適用・引用にあたっては
> 一次資料および最新の学術的議論を確認してください。

## 採録の原則

1. **権威ある出典での確証** — 独立した2出典以上、うち1件以上は大学・学会・査読論文・公的機関・百科事典級
   リファレンス・原典/一次資料で確認できた概念のみ採録。確認できないものは破棄。
2. **トレーサビリティ** — 提唱者・初出（年・文献）を `keyFigures` に明示。
3. **中立性** — 学説には批判・異説がありうるため、必要に応じて `statement` に限界・批判も併記。
4. **5分野＋関連** — 経済学 / 経営学 / 人間科学 / ビジネス法務 / 情報社会学 を `discipline` で分類。

## 収集・検証フロー

1. 大学・学会・査読論文・公的機関・百科事典級リファレンス・原典等から収集。
2. 6並列調査エージェントで概念ごとに確証（≥2出典・うち権威ある出典1件以上）。確認できないものは破棄。
3. 確証できた概念のみ `VERIFIED_CONCEPTS` に採録し、本書の表に追記。
