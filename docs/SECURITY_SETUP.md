# Security タブ セットアップ

Service Hub の **Security** タブは、業務効率とセキュリティ可視性を 1 画面に
まとめるための統合ダッシュボードです。3 種類の独立した情報源を集約します:

| 情報源 | 何を見る | 接続方法 |
|---|---|---|
| **Norton 360** | インストール検出のみ（脅威統計やスキャン結果は不可）| 既知パスを `node:fs` でチェック |
| **Have I Been Pwned (HIBP)** | 任意のメールアドレスが既知の漏洩に含まれていないか | 公開 REST API ($3.50/mo の API キー) |
| **VirusTotal** | URL を ~70 のセキュリティエンジンで一斉照会 | 公開 REST API (無料枠あり) |

## Norton 360 連携の制約 — 先に読むこと

Norton 360 には **コンシューマ向けの公開 REST API が存在しません**。NortonLifeLock /
Gen Digital が公開しているのはエンタープライズ管理 API のみで、個人ライセンスでは
使えません。可能な「連携」は本質的に以下のレベルです:

- ✅ **インストール検出** — Windows / Mac の既知のインストールパスを見て、Norton 360 が
  その PC に入っているかどうかを判定
- ❌ リアルタイム脅威カウント取得（公開 API なし）
- ❌ 直近スキャン結果の取得（内部 SQLite/XML を解析しない限り不可、バージョン依存で壊れやすい）
- ❌ スキャンのトリガ（CLI も公開されていない）
- ❌ 定義ファイル鮮度の取得（同上）

このため、Security タブの実用価値の大部分は **HIBP と VirusTotal** で構成されます。
Norton 360 部分は「PC に AV が入っているか」の安心バナー以上のことはしません。

## HIBP API キーの取得

1. https://haveibeenpwned.com/API/Key を開く
2. メールアドレスを登録 → クレジットカードで $3.50/月（個人利用に十分）
3. 表示された 32 文字の `hibp-api-key` をコピー

無料枠はありません（HIBP が SPAM / 大量ダウンロードを防ぐため意図的に有料化）。

## VirusTotal API キーの取得

1. https://www.virustotal.com/gui/sign-up でアカウント作成（無料）
2. アカウント設定 → API Key をコピー
3. 無料枠: 4 リクエスト/分・500/日（個人で URL チェック程度なら十分）

## アプリへの設定

両 API キーは **1 つの JSON 文字列** として Security タブの「API キー設定 (JSON)」
ボタンから保存します:

```json
{"hibp":"32文字のhibpキー","vt":"VirusTotalキー"}
```

片方だけでも有効。例:

```json
{"hibp":"32文字のhibpキー"}
```

→ HIBP のメール漏洩照会のみ使え、URL スキャンはボタンが無効化されます。

## 使い方

### メール漏洩チェック

1. 「メール漏洩チェック」セクションの「チェック」ボタンを開く
2. 自分のメールアドレスを入力 → 「照会」
3. 結果:
   - **✓** = 既知の漏洩に該当なし（HIBP の 404 レスポンスを「クリーン」として扱う）
   - 漏洩あり: 漏洩名 / 日付 / 流出件数 / 漏洩した情報の種類（パスワード, 電話番号, etc.）が一覧表示

業務応用例:
- 重要メンバーのメールを毎月チェックする運用ルールを作る
- 顧客メールリスト更新時に新規追加分を一括チェック（API rate に注意）

### URL スキャン

1. 「URL スキャン」の「スキャン」ボタンを開く
2. 不審な URL（メールに含まれていたリンクなど）を貼り付け → 「実行」
3. 結果:
   - 検出 0/N: 安全と推定
   - 検出 1〜2/N: 警告（フィッシング新種で 1 つだけ反応するケースあり）
   - 検出 3+/N: 開かない方が良い
4. 「詳細レポート」で VirusTotal Web UI の完全な分析結果へ遷移

業務応用例:
- 顧客から届いた請求書 PDF や招待リンクを開く前にチェック
- フィッシングメール疑いを Slack 通報する前のトリアージ

## 自動化・拡張アイデア

このタブの基盤は完成しているので、以下は `src/main/clients/security.ts` に
新しい action を追加するだけで実装可能:

- **ファイルハッシュ照会**: ローカルファイルの SHA256 → VirusTotal で過去スキャン結果検索
- **ドメイン評価**: VT の `/domains/{d}` エンドポイント
- **複数メール一括照会**: バッチ実行 + レート制限の自動バックオフ
- **Norton ログのテキスト抽出**: もし Windows で Norton ログ (`%ProgramData%\Norton\...`) が
  読めれば、grep で「脅威検出」イベントだけ抽出して時系列表示
- **macOS XProtect**: `/Library/Apple/System/Library/CoreServices/XProtect.bundle` の
  定義鮮度を表示（ローカル `plist` を読むだけ）

各々半日以内で実装可能です。
