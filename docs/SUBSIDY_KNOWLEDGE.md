# 補助金・助成金・給付金 知識ベースと確証プロセス（恒久運用）

Service Hub が参照する国・都道府県・市町村の補助金/助成金/給付金の制度情報は、
法務・税務・労務の知識ベース（[`COMPLIANCE_KNOWLEDGE.md`](COMPLIANCE_KNOWLEDGE.md)）と
**同じ確証ディシプリン**で収集・蓄積する。

> ⚠️ 本データは一般的な制度情報であり、**申請可否・金額・締切の保証ではありません**。
> 補助金・助成金は金額・補助率・公募締切が公募回／年度ごとに変動します。申請にあたっては
> 必ず最新の公募要領／支給要領および専門家の確認を要します。

## 原則

1. **確証のとれたもののみ採録** — 独立 2 出典以上、うち公的（`.go.jp` / 公式実施機関）1 件以上で
   確認できた制度のみ `src/renderer/data/subsidyKnowledge.ts` の `VERIFIED_SUBSIDIES` に採録。
2. **変動値は固定しない** — 補助上限額・補助率・引上げ額区分・**公募締切**は公募回／年度で変わるため、
   固定値として断定せず「最新の公募要領／支給要領で要確認」と明記する（古い締切情報は有害）。
3. **網羅より正確性** — 全国約 1,700 市町村すべての制度の網羅・締切維持は精度の観点から行わない。
   制度レベルの正確な情報＋自治体横断の公式検索ポータル（`SUBSIDY_PORTALS`）への導線で補完する。
4. **専門家レビュー必須** — 申請判断は社労士・中小企業診断士等の専門家の確認を前提とする。

## 仕組み（コードで強制）

- `src/renderer/data/subsidyKnowledge.ts`
  - `VERIFIED_SUBSIDIES`（`VerifiedSubsidy[]`）— 制度ごとに `level`（national/prefecture/municipality）・
    `domain`・`name`・`authority`（所管/実施機関）・`statement`・`application`（申請時期・方法）・`sources` を保持。
  - `SUBSIDY_PORTALS` — 国・都道府県・市町村の「募集中・締切付き」制度を公式ソースで探すための一次導線
    （jGrants / ミラサポplus / J-Net21 / 厚労省 雇用関係助成金一覧）。

## 恒久運用フロー（再実行可能）

1. 各種省庁（厚労省・経産省・中小企業庁・国税庁・総務省 等）・各都道府県・各市町村・各種メディアから収集。
2. 6 並列調査エージェントで制度ごとに確証（≥2 出典・うち公的 1 件以上）。確証できないものは破棄。
3. 確証済みを `VERIFIED_SUBSIDIES` に一括追加し、本書の表を同期。CI 全ゲート green を確認して PR。

## 現在の確証済み制度（2026-06 照合・いずれも独立 2 出典以上／公的 1 件以上）

| 区分 | 制度 | 所管/実施機関 | 申請の枠組み（締切・金額は要確認） |
|---|---|---|---|
| 国・労務 | キャリアアップ助成金 | 厚生労働省・労働局 | 事前にキャリアアップ計画提出→取組→支給申請 |
| 国・労務 | 業務改善助成金 | 厚生労働省・労働局 | 計画作成→交付申請（事業実施前）→実施→支給申請 |
| 国・労務 | 人材開発支援助成金 | 厚生労働省・労働局 | 訓練計画の事前届出→訓練実施→支給申請 |
| 国・労務 | 両立支援等助成金 | 厚生労働省・労働局 | 取組実施→労働局へ支給申請（電子申請可） |
| 国・事業 | 小規模事業者持続化補助金 | 中小企業庁・商工会議所/商工会 | 経営計画を商工会議所等の確認→公募回ごとに電子申請（jGrants） |
| 国・事業 | IT導入補助金（→デジタル化・AI導入補助金） | 経産省・中小企業庁・中小機構 | IT導入支援事業者と連携・gBizIDで電子申請・公募回ごとの締切 |
| 国・事業 | ものづくり補助金 | 中小企業庁・全国中小企業団体中央会 | 事業計画策定→gBizIDで電子申請・公募回ごとの締切 |
| 国・事業 | 事業再構築補助金（→中小企業新事業進出補助金に後継） | 中小企業庁・中小機構 | 認定支援機関と計画策定→電子申請・公募回ごとの締切（新規公募は要確認） |
| 国・労務 | 特定求職者雇用開発助成金 | 厚生労働省・労働局 | ハローワーク紹介で就職困難者を継続雇用→支給対象期ごとに申請 |
| 国・労務 | トライアル雇用助成金 | 厚生労働省・ハローワーク | ハローワーク紹介で試行雇用→計画書提出→終了後に支給申請 |
| 国・労務 | 65歳超雇用推進助成金 | 厚労省・JEED | 定年引上げ等の措置実施→JEED都道府県支部へ申請 |
| 都道府県・事業 | 創業助成事業（創業助成金）（東京都・代表例） | 東京都・東京都中小企業振興公社 | 年度ごとの公募制→電子申請（jGrants） |

## 自治体横断の公式検索ポータル（最新の締切付き情報はここで確認）

- **jGrants**（デジタル庁）— 募集中の補助金の検索・電子申請
- **ミラサポplus**（中小企業庁）— 国・自治体の支援制度・補助金検索
- **J-Net21 支援情報ヘッドライン**（中小機構）— 補助金・助成金・融資の最新情報
- **厚生労働省 雇用関係助成金**（一覧・支給要領）

公募要領 URL・申請窓口は `subsidyKnowledge.ts` の各 `sources` に機械可読で保持。改正・公募更新は本書と当該データを PR で更新する。
