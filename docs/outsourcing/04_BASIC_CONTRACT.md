# 業務委託基本契約書（ソフトウェア開発）テンプレート

> ⚠️ 本書は一般的な雛形であり法的助言ではない。**締結前に必ず弁護士のレビュー**を受けること。
> 請負に該当する個別契約は印紙税の課税文書（第2号文書）となる場合がある。
> 下請代金支払遅延等防止法（下請法）の適用がある取引では、書面交付義務・支払期日
> （受領後60日以内）等を遵守すること。

---

（発注者名）（以下「甲」という）と（受託者名）（以下「乙」という）とは、甲が乙に委託する
ソフトウェア開発業務（以下「本件業務」という）に関し、次のとおり基本契約（以下「本契約」
という）を締結する。

### 第1条（目的・適用）
1. 本契約は、甲乙間の本件業務に関する個別契約（以下「個別契約」という）に共通して
   適用される基本的事項を定める。
2. 個別契約は [05_SOW.md](./05_SOW.md) の様式により締結し、本契約と個別契約の定めが
   抵触するときは個別契約が優先する。

### 第2条（契約形態）
1. 個別契約は、その性質に応じ次のいずれかの形態を明記して締結する。
   (1) **請負**: 乙は成果物を完成させ甲に納入する義務を負う（開発委託に適用）。
   (2) **準委任**: 乙は善良な管理者の注意をもって業務を遂行する義務を負う
       （調査・保守・技術支援に適用）。

### 第3条（仕様・開発標準）
1. 成果物の仕様は、個別契約および甲が提示する要件定義書・設計書・仕様書
   （`docs/specs/` 一式）による。
2. 乙は、甲が指定する開発標準に従う。少なくとも次を含む。
   (1) 開発フロー: フィーチャーブランチ → Pull Request → 甲レビュー → マージ。main 直接 push の禁止
   (2) 品質ゲート: 個別契約に定める自動検証（typecheck / テスト / verify:all / mutation / build）の全グリーン維持
   (3) 安全要件: AIカウンセラーの安全不変条件（specs/05 §7）に関わる変更は甲の書面承認を要する
   (4) セキュリティ: 別紙セキュリティ・データ保護要件書（[06_SECURITY_REQUIREMENTS.md](./06_SECURITY_REQUIREMENTS.md)）の遵守

### 第4条（再委託）
1. 乙は、甲の事前の書面承諾なく本件業務の全部または一部を第三者に再委託してはならない。
2. 承諾を得て再委託する場合、乙は再委託先に本契約と同等の義務を負わせ、
   再委託先の行為について一切の責任を負う。

### 第5条（納入・検収）
1. 乙は、個別契約に定める期日までに成果物を納入する。
2. 甲は、納入後、個別契約に定める検収期間内に、検収基準書
   （[09_ACCEPTANCE.md](./09_ACCEPTANCE.md)）に基づき検査を行い、合否を乙に通知する。
3. 不合格の場合、乙は甲乙協議のうえ定めた期限までに無償で修補し、再度検査を受ける。
4. 検収期間内に甲が合否の通知をしないときは、期間満了をもって検収に合格したものとみなす。

### 第6条（委託料・支払）
1. 委託料および支払条件は個別契約に定める。
2. 甲は、検収合格後、乙の請求書受領月の翌月末日までに委託料を支払う
   （下請法適用時は成果物受領後60日以内の支払期日を厳守する）。

### 第7条（知的財産権）
1. 成果物に関する著作権（著作権法第27条・第28条の権利を含む）は、委託料の完済時に
   乙から甲に譲渡される。ただし、乙または第三者が従前から保有する著作物
   （汎用ライブラリ・ツール等）の権利は乙または当該第三者に留保され、乙は甲に対し
   成果物の利用に必要な範囲で無償の利用を許諾する。
2. 乙は、甲および甲の指定する者に対し著作者人格権を行使しない。
3. 成果物に OSS を含める場合、乙はそのライセンス（MIT / Apache-2.0 等）と義務を
   納品時に一覧で明示し、甲の利用目的と矛盾しないことを確認する。
4. 成果物が第三者の知的財産権を侵害した場合、乙はその責任と費用で解決し、
   甲に生じた損害を賠償する。

### 第8条（契約不適合責任）
1. 検収合格後、成果物に種類・品質が契約の内容に適合しないこと（契約不適合）が
   発見されたときは、甲は乙に対し修補による履行の追完を請求できる。
2. 前項の請求は、検収合格後**1年以内**に甲が乙に通知した場合に限る。
3. 契約不適合が乙の責に帰すべき事由による場合、甲は損害賠償を請求できる。

### 第9条（秘密保持・個人情報）
甲乙間で締結した秘密保持契約（[02_NDA.md](./02_NDA.md)）は本件業務にも適用される。
個人情報の取扱いは同契約第5条および別紙セキュリティ・データ保護要件書による。

### 第10条（損害賠償）
甲および乙は、本契約または個別契約の履行に関し相手方の責に帰すべき事由により損害を
被ったときは、相手方に対し損害賠償を請求できる。ただし、賠償額は当該個別契約の
委託料額を上限とし、逸失利益・間接損害を含まない。故意または重過失による場合は
この限りでない。

### 第11条（解除)
1. 甲または乙は、相手方が次の各号のいずれかに該当したときは、催告なく本契約および
   個別契約の全部または一部を解除できる。
   (1) 重大な契約違反があり、相当期間を定めた催告後も是正されないとき
   (2) 支払停止・破産・民事再生・会社更生等の申立てがあったとき
   (3) 反社会的勢力に該当し、または関与が判明したとき
2. 解除は損害賠償の請求を妨げない。

### 第12条（反社会的勢力の排除）
甲および乙は、自己およびその役員が反社会的勢力に該当しないことを表明し、将来に
わたって関与しないことを確約する。

### 第13条（有効期間）
本契約の有効期間は締結日から1年間とし、期間満了の1か月前までに申し出がないときは
同一条件で1年間更新され、以後も同様とする。本契約終了時に有効な個別契約については、
当該個別契約の終了まで本契約が適用される。

### 第14条（協議・管轄）
1. 本契約に定めのない事項または疑義は、甲乙誠実に協議して解決する。
2. 本契約に関する紛争は、（管轄裁判所名）地方裁判所を第一審の専属的合意管轄裁判所とする。
