# 01. 資金調達の全体像

## A. 4 種の本質的な違い

### A-1. 補助金 (Hojokin)

- **性質**: 国 / 自治体の **政策誘導手段**。特定の投資・取組を促すために予算を割り当て、競争的に配分
- **返済**: **不要 (原則)** だが、要件未達 / 不正 / 目的外利用で **返還命令**
- **タイミング**: **後払い** (実施 → 報告 → 入金) が基本。実施期間中の資金繰りはユーザー負担
- **採択率**: 制度・年度により 10〜70% と大きな幅
- **典型額**: 数十万 〜 数億円
- **税務**: 受給時に**益金算入**(課税対象)。圧縮記帳で課税繰延可能なケースあり

### A-2. 助成金 (Joseikin)

- **性質**: 主に **厚労省所管**、雇用関連の労働政策ツール
- **返済**: 不要
- **タイミング**: 後払い、ただし要件達成すれば**ほぼ確実に支給**
- **特徴**: 競争性なく、要件主義。書類を整え要件を満たせば受給可能
- **落とし穴**: **雇用維持義務** (例: 助成対象者を 6 ヶ月以上雇用継続)・**制度変更による期中の縮減**

### A-3. 公庫融資 (日本政策金融公庫)

- **性質**: 政府 100% 出資の政策金融機関による **融資**
- **3 事業**: 国民生活事業 (個人事業・小規模) / 中小企業事業 / 農林水産事業
- **特徴**: 民間が貸さないステージ・用途にも貸す (起業直後・赤字局面)
- **金利**: 民間より低め (基準金利 〜 特別利率)
- **保証**: ケースによる (新創業融資制度は無担保・無保証人 等)
- **返済**: 必要。元金据置あり

### A-4. 民間金融機関融資

- **メガバンク** / **地方銀行** / **信用金庫** / **信用組合** / **ノンバンク** / **ファクタリング** ……
- **特徴**: 業歴・実績・担保・保証協会付き保証 で判断
- **金利**: 公庫より高めだが、関係構築で次回以降のスピード向上
- **保証協会**: 信用保証協会の保証付き (責任共有制度)。万一返済不能時は保証協会が代位弁済 → ユーザーには協会への返済義務が残る

## B. 比較マトリクス

| 観点 | 補助金 | 助成金 | 公庫融資 | 民間融資 |
|---|---|---|---|---|
| 返済 | 不要 | 不要 | 要 | 要 |
| 競争性 | 高 | 低 (要件) | 中 | 中〜高 |
| 入金時期 | 事業終了後 | 要件達成後 | 申込から数週 | 数日〜数週 |
| 自己資金必要性 | 高 (立替) | 中 | 低〜中 | 中〜高 |
| 担保 / 保証 | 不要 | 不要 | 制度による | 通常必要 |
| 計画書 / 申請書類量 | 多 | 中 | 中 | 中〜少 |
| 採択 / 承認の不確実性 | 高 | 低 | 中 | 中 |
| 不正時のペナルティ | 重 (返還+加算金+刑事) | 重 | 期限の利益喪失 | 期限の利益喪失 |

## C. 用途別の典型

| やりたいこと | 第一候補 | 補完 |
|---|---|---|
| 設備投資 (機械導入) | ものづくり補助金 | 公庫設備資金 |
| IT ツール導入 | IT 導入補助金 | 公庫 |
| 販路開拓・販促 | 小規模事業者持続化補助金 | 民間運転資金 |
| 新分野進出・業態転換 | 後継的「事業再構築」枠 | 公庫資本性ローン |
| 起業 | 新創業融資 (公庫) | 創業補助金 (自治体) |
| 雇用 (正社員化等) | キャリアアップ助成金 | — |
| 雇用維持 | 雇用調整助成金 | 運転資金融資 |
| 事業承継・M&A | 事業承継・引継ぎ補助金 | 公庫事業承継融資 |
| 研究開発 | NEDO 等 R&D 助成 | 公庫 |
| 海外展開 | JETRO 等 | 公庫国際協力 |

## D. 経営戦略との関係

```
[1. 経営者のビジョン]
        ↓
[2. 中期経営計画 (3〜5 年)]
        ↓
[3. 単年度の実行計画]
        ↓ (ここで初めて資金調達の議論)
[4. 必要資金の算定]
   ├── 設備投資 → 補助金 (ものづくり) + 公庫設備
   ├── IT 投資 → IT 導入 + 公庫
   ├── 人材投資 → 助成金 (キャリアアップ) + 民間運転
   └── 運転資金 → 民間 (短期) / 公庫 (長期)
        ↓
[5. 資金繰り表 (月次 24 ヶ月)]
        ↓
[6. 申請 / 借入実行]
        ↓
[7. 実施 → モニタリング → 完了報告 → 入金 / 返済]
```

戦略と資金は不可分。本書群はこの流れを **AI 支援で効率化** する。
