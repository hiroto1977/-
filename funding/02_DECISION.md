# 02. 判断フレームワーク

「自分の場合、何を使うべきか」を最短で絞り込むためのフロー集。

## A. メイン決定木

```
                       資金調達の必要性が出た
                              │
        ┌─────────────────────┼──────────────────────┐
        │                     │                      │
   返済不要 ?              ある程度の               大きな
   絶対欲しい                自己資金あり           借入が必要
        │                     │                      │
   ┌────┴────┐                │                      │
   │         │                ↓                      ↓
 雇用関係?  設備/IT?      民間融資 (短期)           公庫融資
   │         │            金利低・関係構築          (長期・低利)
   ↓         ↓                +

```

## B. 状況別 推奨パターン (10 ケース)

### ケース 1: 起業直後・自己資金少
```
公庫「新創業融資制度」(無担保・無保証) → ※ 制度の最新条件を必ず公庫窓口で確認
+ 自治体の創業補助金
+ 創業時の助成金 (要件あれば)
NG: 民間銀行の無担保融資 (実績なし)
```

### ケース 2: 業歴 3 年・黒字・設備投資したい
```
ものづくり補助金 (設備の 1/2 を補助) を出願
  ↓ 採択を待ちつつ
公庫の設備資金 + 民間長期運転 で実施
  ↓ 補助金入金
公庫 / 民間の繰上返済 を検討
```

### ケース 3: 業歴 10 年・赤字 1 期・運転資金が逼迫
```
公庫の経営改善貸付 (マル経) ※商工会議所経由
+ 民間: 信用保証協会の経営安定資金
+ 雇用維持なら 雇用調整助成金
NG: 補助金 (後払いで間に合わない)
ファクタリング・ノンバンクは最終手段 (高金利)
```

### ケース 4: 新分野進出・業態転換したい
```
事業再構築 (後継) 系の補助金 — 計画は中小企業診断士と
+ 公庫の資本性ローン (劣後ローン: 純資産扱い)
+ 自己資金確保
注: 新規事業の売上計画は厳しめに見る
```

### ケース 5: 正社員を増やしたい
```
キャリアアップ助成金 (正社員化コース)
+ 採用関連の自治体助成 (地域による)
+ 必要なら賃金上昇分の運転資金 (公庫 or 民間)
```

### ケース 6: IT ツールを導入したい
```
IT 導入補助金 (通常 / セキュリティ対策推進 / インボイス対応 等の枠)
+ 残額は自己資金 or 短期民間融資
注: IT ベンダ (IT 導入支援事業者) との二人三脚必須
```

### ケース 7: 販路開拓・新商品 PR したい
```
小規模事業者持続化補助金 — 商工会議所/商工会の伴走
+ 必要に応じ公庫
注: 補助対象経費の限定 (人件費は対象外 等) を要確認
```

### ケース 8: 親族外 M&A で会社を譲り受ける
```
事業承継・引継ぎ補助金 (M&A 専門家活用枠 等)
+ 公庫の事業承継・集約・活性化支援資金
+ 民間バンクの M&A ローン
+ デューデリジェンスは弁護士・税理士・会計士・FA を起用
```

### ケース 9: コロナ等で売上急減・雇用維持したい
```
雇用調整助成金 (時限的拡大は要確認)
+ 公庫のセーフティネット貸付
+ 民間: 信用保証協会のセーフティネット保証
```

### ケース 10: 研究開発のシーズあり、製品化までは時間
```
NEDO / AMED 等の R&D 助成 (テーマ募集型・公募型)
+ 大学・公的研究機関との共同研究 → マッチング助成
+ 公庫の新事業活動促進資金
+ ベンチャーキャピタル / 中小企業基盤整備機構ファンド
```

## C. NG パターン早見表

| 症状 | 何が悪いか | 対処 |
|---|---|---|
| 補助金 通ったから設備買おう | 後払いで 数ヶ月分 立替必要 | つなぎ融資の事前確保 |
| 助成金 30 万円のために週 40 時間の労務管理 | コスト > リターン | 助成金額/手間 を試算 |
| 公庫と民間に同じ事業計画 (内容矛盾) | 信用毀損 | 計画は 1 本に統一 |
| 補助金併用で同じ経費を 2 重請求 | **不正受給** | 経費区分を厳格管理 |
| 採択前に発注 | **対象外** | 公募要領の交付決定前ルール厳守 |
| 申請書を AI に丸投げ | 公募趣旨と乖離・採択後遂行不能 | 専門家レビュー必須 |
| 数値の水増し (売上見込・雇用人数) | **詐欺罪** | 根拠ある合理的見込みのみ |
| 制度の趣旨と異なる用途に流用 | **目的外使用 → 返還命令** | 計画通り実施 |

## D. 「使わない」判断もある

以下の場合、補助金 / 助成金 の利用は **避けるか慎重に**:
- 申請書作成に貴重な経営者時間を奪われ、本業が止まる
- 助成対象人材の雇用維持義務が、後の事業縮小の足かせになる
- 補助金の事業実施期間に縛られて、市場変化への即応性を失う
- 報告事務の継続コスト > 補助金額

> **判断軸**: 「補助金がなくてもやるべき投資か?」が Yes なら申請、No なら申請しない (補助金につられた事業は破綻しやすい)。

## E. 専門家の使い分け

| 目的 | 推奨専門家 | 報酬目安 (個別差大) |
|---|---|---|
| 補助金申請書 (戦略部分) | 中小企業診断士 | 着手金 + 成功報酬 (採択額の 5〜15%) |
| 助成金申請 (労務要件) | 社会保険労務士 | 着手金 + 受給額の 10〜20% |
| 公庫融資 (面談含む) | 中小企業診断士 / 金融機関出身者 | プロジェクト単位 |
| 民間銀行交渉 | 元銀行員のコンサル / 顧問税理士 | 月額 or 単発 |
| 事業計画 (財務) | 公認会計士 / 税理士 | 月額 顧問 + 計画作成費 |
| 法務 (契約・補助金規程確認) | 弁護士 | 時間制 or 顧問 |

複数専門家を組み合わせるが、**司令塔は経営者本人**。

## F. 自己診断 12 問

```
1. □ 直近 3 年の決算書を 30 分で説明できる
2. □ 月次資金繰り表を作成・更新している
3. □ 売上の主要 KPI を 3 つ即答できる
4. □ 競合 3 社の強み / 弱みを言える
5. □ 自社の中期計画 (3 年) を 1 枚にまとめている
6. □ 補助金実施の 6 ヶ月後の月次収支を見積もれる
7. □ 借入残高 / 月商比率を把握している
8. □ 主要金融機関の担当者の名前を覚えている
9. □ 商工会議所 / 商工会に登録している
10. □ 顧問税理士 (or CFO 機能) がいる
11. □ 直近で公的支援機関 (よろず支援拠点 等) を訪問した
12. □ 不正受給した場合の罰則を答えられる
```

8 問以下: 申請前に経営の足元固めから。9〜11 問: 個別補助金は出願可。12 問: 全制度を戦略的に組み合わせ可。
