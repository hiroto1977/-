# 03. 主要プログラム カタログ

> ⚠️ 制度名・補助率・上限額・公募時期は **毎年変動する**。本書は「概念」「対象範囲」「典型的姿」のみ記載。
> **最新の公募要領を必ず一次情報で参照** すること。一次情報リンクは末尾。

## A. 補助金 (経産省 / 中小企業庁系)

### A-1. ものづくり・商業・サービス生産性向上促進補助金 (通称: ものづくり補助金)

| 項目 | 内容 |
|---|---|
| 主旨 | 中小企業の革新的な製品・サービス開発・生産プロセス改善 |
| 主な対象 | 機械装置・システム購入、外注費、技術導入費 |
| 補助率 | 1/2 〜 2/3 (枠による) |
| 補助額 | 数百万円 〜 数千万円規模 |
| 必要書類 | 事業計画書 (10〜20 ページ)、見積書、決算書 |
| 採択基準 | 革新性・事業化可能性・政策面 |
| 注意点 | 採択 → **交付決定** → 発注 (順序厳守)、生産性向上計画達成義務 (賃上げ要件あり) |

### A-2. 小規模事業者持続化補助金

| 項目 | 内容 |
|---|---|
| 主旨 | 小規模事業者の販路開拓・業務効率化 |
| 主な対象 | 広報費、ウェブ構築、展示会、機械装置 (一部) |
| 補助率 | 2/3 程度 |
| 補助額 | 数十万円 〜 数百万円 |
| 申請ルート | **商工会議所 / 商工会 経由** (経営計画書を含む) |
| 注意点 | 小規模事業者 (常時従業員数の業種別上限) のみ対象 |

### A-3. IT 導入補助金

| 項目 | 内容 |
|---|---|
| 主旨 | 中小企業の IT ツール導入支援 |
| 主な対象 | ソフトウェア、クラウドサービス、PC・タブレット (枠による)、セキュリティ |
| 枠 | 通常枠、セキュリティ対策推進枠、インボイス対応枠 等 (枠は再編される) |
| 補助率 | 1/2 〜 3/4 |
| 補助額 | 数十万円 〜 数百万円 |
| 注意点 | **IT 導入支援事業者 (登録ベンダ)** との連名申請。事業者選定が成否を分ける |

### A-4. 事業承継・引継ぎ補助金

| 項目 | 内容 |
|---|---|
| 主旨 | 事業承継・M&A による引継ぎを支援 |
| 主な対象 | 経営革新枠 / 専門家活用枠 / 廃業・再チャレンジ枠 |
| 補助率 | 1/2 〜 2/3 |
| 注意点 | M&A の DD 費用・FA 報酬等が対象になる枠あり |

### A-5. 事業再構築 後継的補助金

> 「事業再構築補助金」自体は時限的措置として運営されたが、後継・関連する **新分野進出 / 業態転換系** の補助金が継続。最新の公募要領で名称・要件を確認。

### A-6. 各自治体の創業補助金 / 設備投資補助金

| 例 | 概要 |
|---|---|
| 東京都 創業助成事業 | 東京都内で創業 5 年未満等の要件 |
| 各市区町村 商店街活性化 | 地域限定 |
| 地方版総合戦略の各種補助金 | 移住・定住・地域産業振興 |

→ **自治体公式サイト** + 商工会議所への問い合わせが基本ルート。

## B. 助成金 (厚労省系)

### B-1. キャリアアップ助成金

| 項目 | 内容 |
|---|---|
| 主旨 | 非正規雇用労働者のキャリアアップ |
| 主なコース | 正社員化コース、賃金規定改定コース、賃金規定共通化コース、賞与・退職金制度導入コース、社会保険適用時処遇改善コース 等 |
| 受給額 | コース・要件により数十万円 |
| 必要 | 就業規則改定、キャリアアップ計画書、有期雇用契約からの転換証明 等 |
| 注意点 | **キャリアアップ計画書の事前提出**、転換後の継続雇用 6 ヶ月以上、賃上げ要件 |

### B-2. 雇用調整助成金

| 項目 | 内容 |
|---|---|
| 主旨 | 経済上の理由で休業等を行う事業主に休業手当の一部を助成 |
| 受給額 | 1 人 1 日あたり上限あり |
| 注意点 | **休業実施 計画届** の事前提出、不支給要件 (生産指標確認、解雇しない 等) |

### B-3. 両立支援等助成金

| 項目 | 内容 |
|---|---|
| 主旨 | 出生時両立支援 / 介護離職防止 / 育児休業等 |
| 注意点 | コース別に詳細要件、就業規則の整備が前提 |

### B-4. 人材確保等支援助成金

| 項目 | 内容 |
|---|---|
| 主旨 | 雇用管理改善、設備改善、テレワーク導入 等 |
| 注意点 | 計画届 → 実施 → 効果測定 → 申請のサイクル |

### B-5. その他の主要助成金

| 名称 | 概要 |
|---|---|
| 人材開発支援助成金 | 教育訓練の費用・賃金助成 |
| トライアル雇用助成金 | 就業経験不足者等の試用期間に助成 |
| 特定求職者雇用開発助成金 | 高齢者・障害者・母子家庭の母 等の雇用 |
| 業務改善助成金 | 最低賃金引上げ + 設備投資 |

> 詳細は **厚労省ハローワーク** または **社労士** を経由するのが鉄則。

## C. 公庫融資 (日本政策金融公庫)

### C-1. 国民生活事業 (個人事業・小規模事業者向け)

| 制度 | 概要 |
|---|---|
| **新創業融資制度** | 創業 (新規開業前 / 開業後) 向け、無担保・無保証人での融資制度 (条件あり) |
| 一般貸付 | 業種ほぼ問わず運転・設備資金 |
| 経営環境変化対応資金 (セーフティネット貸付) | 売上減少・取引先倒産 等の急場 |
| 小企業等経営改善資金 (マル経融資) | **商工会議所/商工会 の推薦** が必要、無担保・無保証 |
| 教育一般貸付 | 個人向け教育費 (事業用ではないが関連あり) |
| 食品貸付・生活衛生貸付 | 業種特化 |

### C-2. 中小企業事業 (中規模事業者向け)

| 制度 | 概要 |
|---|---|
| 新事業活動促進資金 | 新分野進出・第二創業 |
| 企業活力強化資金 | 設備・システム投資 |
| 環境・エネルギー対策資金 | 省エネ・再エネ投資 |
| **資本性ローン (挑戦支援資本強化特別貸付)** | 純資産扱いの劣後ローン。財務改善効果 |
| 海外展開・事業再編資金 | 海外進出・再編 |

### C-3. 農林水産事業

農業・林業・漁業・食品関連の専門融資。地域金融機関や JA との連携が一般的。

### C-4. 共通の特徴

- **金利**: 基準利率 (年に何度か改定) + 各特別利率 (省エネ・女性若者シニア・新規開業 等で軽減)
- **据置期間**: 元金返済を 1〜5 年程度据え置けるケースあり
- **保証人**: 制度により異なる
- **準備書類**: 借入申込書、創業計画書 / 企業概要書、設備見積書、決算書 (3 期分)、試算表、納税証明、本人確認 等

## D. 民間金融機関融資

### D-1. プロパー融資 vs 保証協会付

| | プロパー | 保証協会付 |
|---|---|---|
| 担保 | 信用 + 物的担保 | 信用保証協会の保証 |
| 金利 | 低め | + 保証料 |
| 銀行のリスク | 100% | 部分的 (責任共有制度) |
| 借りやすさ | 業歴・実績必要 | 創業・小規模に有効 |

### D-2. 制度融資 (自治体 + 信用保証協会 + 金融機関)

各都道府県・市区町村が **斡旋・利子補給・保証料補助** を行う融資制度。

- 例: 東京都中小企業制度融資 「事業一般」「創業」「小規模零細企業」「セーフティネット」 等
- 自治体窓口で斡旋書を取得 → 金融機関へ

### D-3. 信用金庫 / 信用組合

地域密着型。創業期や小規模事業者には **メガバンクより使いやすい**。

### D-4. 短期 vs 長期

| | 短期 (1 年以内) | 長期 (1 年超) |
|---|---|---|
| 用途 | 季節資金、つなぎ資金 | 設備、長期運転 |
| 金利 | 通常低い | 高め |
| 借換 | 手形ジャンプ等 | 計画的償還 |

## E. その他の調達手段

### E-1. リース / 割賦

設備調達を借入の代替で。所有権はリース会社、税務上の扱いに注意。

### E-2. ファクタリング

売掛債権の売却。手数料は高め。最終手段。

### E-3. クラウドファンディング

- 購入型 (Makuake / CAMPFIRE 等): 商品の先行販売型
- 寄付型: 公益性ある事業
- 株式投資型 (FUNDINNO 等): 第二種金融商品取引業の制限あり
- 融資型 (ソーシャルレンディング): 過去にトラブル多発、要慎重評価

### E-4. ベンチャーキャピタル

成長性のあるスタートアップ向け。**経営権の希薄化** + **EXIT 圧力** を伴う。

### E-5. 中小機構 (中小企業基盤整備機構) ファンド

公的色のあるファンド。中小機構経由の各種ファンド。

### E-6. 寄付金・公的資金 (企業版ふるさと納税 等)

地域連携プロジェクトで活用余地。
