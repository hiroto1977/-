# 05. 事業計画書 ひな形

補助金・融資・経営計画 すべての一次資料。**1 本の事業計画書を、用途別に体裁調整して使い回す** のが効率的 (内容矛盾のリスクも回避)。

## A. 共通フォーマット (15 セクション)

### 1. 表紙
- 計画名
- 申請者 (法人名 or 屋号 + 代表者氏名)
- 業種コード
- 作成日 / 改訂版数
- 連絡先

### 2. エグゼクティブ サマリー (1 ページ)
- 事業の本質 (1 行)
- 解決する課題 (1 行)
- ソリューション (3 行)
- ターゲット顧客 (1 行)
- 収益モデル (1 行)
- 必要資金 / 調達方針 (1 行)
- 3 年後の姿 (KPI 3 つ)

### 3. 事業者概要
- 沿革
- 株主構成 / 出資比率
- 役員・従業員数
- 主要取引先 / 仕入先
- 直近 3 期の業績ハイライト

### 4. 経営理念・ビジョン
- ミッション (なぜ存在するか)
- ビジョン (どこへ向かうか)
- バリュー (何を大切にするか)

### 5. 市場分析
- 市場規模 (TAM / SAM / SOM)
- 市場成長率と背景要因 (PEST)
- 主要トレンド
- 競合分析 (3〜5 社の比較表)

### 6. 顧客分析
- ペルソナ (年齢・職業・課題・購買力)
- カスタマージャーニー
- 既存顧客の声 (アンケート・インタビュー)

### 7. 自社の強み・課題 (SWOT)

```
| 内部 | Strengths            | Weaknesses          |
|      | 強み                  | 弱み                 |
| 外部 | Opportunities        | Threats             |
|      | 機会                  | 脅威                 |
```

掛け合わせ戦略 (SO / WO / ST / WT) も明示。

### 8. 事業内容
- 提供する商品・サービス
- 開発・調達体制
- 知的財産 (特許・商標)
- 競争優位の源泉

### 9. ビジネスモデル
- 収益モデル (単発/継続/従量/サブスク)
- 価格設定根拠
- チャネル戦略
- パートナー戦略

### 10. マーケティング・営業計画
- 4P (Product / Price / Place / Promotion)
- 顧客獲得コスト (CAC) / 顧客生涯価値 (LTV)
- KPI (PV→CV 率, 月間獲得件数 等)

### 11. 実施スケジュール (Gantt)

```
Year   Q1  Q2  Q3  Q4
2026   ■■■                準備
       ■■■■■              開発
            ■■■■           PoC
                ■■■■       本格展開
2027        ■■■■■■■■■■    継続運営
            ■■             成果測定
```

### 12. 組織計画
- 現在の組織図
- 計画後の組織図
- 主要メンバー (経歴・専門性)
- 採用計画 / 教育計画

### 13. 財務計画 (詳細は `06_FINANCIAL_MODEL.md` 参照)
- 売上計画 (3〜5 年)
- 費用計画
- 投資計画
- 損益見通し
- キャッシュフロー
- 資金調達計画

### 14. リスク と 対策
- 市場リスク
- 競合リスク
- オペレーションリスク
- 財務リスク
- 法務リスク
- 各リスクへの対応策

### 15. 補足資料
- 主要顧客契約書 (要点)
- 特許・商標
- 推薦状 / 取引先証明
- 関連プレス記事

## B. 用途別の調整

### B-1. 補助金用 (例: ものづくり補助金)

追加で強調:
- **革新性**: 業界・地域における新規性
- **政策との整合**: 賃上げ・GX/DX・スタートアップ支援 等の政策テーマ
- **付加価値額の向上目標**: 3 年で X% 以上 等
- **遂行可能性**: 体制・実績・スケジュール

不要 / 簡略化可:
- 株主構成 (中小企業の場合)

### B-2. 融資用 (公庫 / 民間)

追加で強調:
- **返済原資**: 営業利益 + 減価償却 が毎月返済額を超える試算
- **経営者の経歴**: 個人保証の人物面審査
- **資金使途の妥当性**: 設備見積・在庫の必要性
- **返済シミュレーション**: 元金均等 / 元利均等の比較

不要 / 簡略化可:
- 詳細マーケティング 4P

### B-3. 助成金用 (キャリアアップ等)

追加で強調:
- **就業規則の整備状況**
- **対象者の選定基準**
- **取組後の継続雇用計画**

不要:
- 市場分析 (一部不要)

### B-4. 中期経営計画 (内部用)

追加で強調:
- **アクションプラン** (誰がいつ何を)
- **KPI と目標値**
- **月次レビュー会議体**

## C. 書き方の原則 (10 か条)

1. **数字を伴う具体性** — 「成長します」ではなく「2 年で 1.5 倍」
2. **第三者検証可能性** — データ出典を明記
3. **読み手 = 審査員 / 銀行員** を想定 — 専門用語は補足
4. **Why から始める** — What / How の前に Why
5. **逆算思考** — Goal → 必要な現在のアクション
6. **過剰な美辞麗句を排する** — 「画期的」「革命的」は避ける
7. **失敗時のプラン B** — リスクへの誠実な向き合い
8. **計画は更新するもの** — 策定 → 月次レビュー → 改訂
9. **AI 出力をそのまま提出しない** — 必ず人間の手で磨く
10. **採択 / 承認だけが目的ではない** — 事業遂行のための地図

## D. 一般的な NG 表現

| NG | 理由 | 言い換え |
|---|---|---|
| 「業界初の革新的な…」 | 検証不可、テンプレ的 | 「2026 年現在 X が成し得ていない Y を…」 |
| 「○○億円市場の獲得を目指す」 | 大きすぎて非現実的 | 「初年度 ○○ 万円、3 年後 ○○ 千万円」 |
| 「絶対に成功させます」 | 主観・根拠なし | 「○○ の理由で達成可能と考える」 |
| 「資金調達できれば全て解決」 | 経営の質を疑われる | 「○○ の課題には ○○ で対応する」 |
| 「AI で○○を…」 (具体性なし) | バズワード | 「○○モデルで○○を実現する」 |
| 「他社にはない我々独自の…」 | 検証なしの主張 | 「特許○○ により他社が模倣困難な…」 |

## F. 計画書 1 本作成の標準工程 (5〜10 営業日)

```
Day 1〜2: 情報収集・自己分析・市場調査
Day 3〜4: 構成案・主要数値の決定 (経営者本人主導)
Day 5〜6: 各セクションのドラフト (AI 補助 OK、ローカル LLM 推奨)
Day 7   : 財務モデル (06_FINANCIAL_MODEL.md) と整合確認
Day 8   : 専門家レビュー (中小企業診断士・税理士 等)
Day 9   : 修正・図表ブラッシュアップ
Day 10  : 最終チェック (E-1 のチェックリスト)・提出
```
