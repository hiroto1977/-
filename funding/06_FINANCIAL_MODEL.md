# 06. 財務モデル (3〜5 年)

事業計画書の数値裏付け。**Excel / Numbers / Google Sheets** で作成。
本書は構造のみ示し、実装は表計算ソフトで。

## A. 必須シート構成

| シート | 内容 |
|---|---|
| 1. **前提条件** | 顧客単価・成長率・人件費単価・税率 等 (1 箇所で管理) |
| 2. **売上計画** | 月次 36〜60 ヶ月 |
| 3. **人件費計画** | 月次、職種別、社保込み |
| 4. **その他経費** | 月次、固変別 |
| 5. **設備投資** | 取得・減価償却スケジュール |
| 6. **資金調達** | 調達源・時期・金額・条件 |
| 7. **借入・返済** | 元利金スケジュール |
| 8. **損益計算書 (PL)** | 月次 → 年次集計 |
| 9. **貸借対照表 (BS)** | 期末残高 |
| 10. **キャッシュフロー (CF)** | 月次・営業/投資/財務 |
| 11. **資金繰り表** | 月次入出金 |
| 12. **KPI ダッシュボード** | 売上・利益・粗利率・社員 1 人あたり 等 |
| 13. **感度分析** | 売上 ±20%, 売価 ±10%, 人件費 +10% |
| 14. **ストレスシナリオ** | ベース / 楽観 / 悲観 |
| 15. **指標 (補助金 / 融資 用)** | 付加価値額・労働生産性・自己資本比率 等 |

## B. 売上計画の作り方

### B-1. ボトムアップ (推奨)

```
売上 = 顧客数 × 単価 × 購入頻度

それぞれを月次で積上げ:
- 顧客数 = 既存継続 + 新規獲得 - 解約
- 単価 = 商品ミックスの加重平均
- 購入頻度 = 購買サイクル
```

### B-2. トップダウン (検算用)

```
TAM (全体市場) → SAM (到達可能) → SOM (獲得目標市場) → 自社シェア
→ 売上見込み
```

両方で算出し、乖離を説明できるようにする。

### B-3. 売上計上タイミング

- 物販: 引渡時
- サービス: 提供完了時
- 月額サブスク: 月末按分 (実現主義)
- 補助金収入: 交付決定時 or 受領時 (会計方針による)

## C. 費用計画

### C-1. 変動費 (売上連動)

- 原材料費、商品仕入
- 外注費 (生産連動)
- 決済手数料
- 配送費 (出荷連動)

### C-2. 固定費 (毎月)

- 役員報酬
- 給与・賞与・退職金引当
- 法定福利費 (健保・厚年・労保)
- 家賃
- 通信費・水道光熱費
- 支払利息
- 減価償却費
- リース料

### C-3. 一時費用 (期初・特定月)

- 設備購入
- 採用費
- 広告キャンペーン
- システム導入費

## D. 設備投資 と 減価償却

```
減価償却費 = 取得価額 ÷ 耐用年数 (定額法)
or
減価償却費 = 期首簿価 × 償却率 (定率法)

少額減価償却資産: 30 万円未満は全額損金算入可 (中小特例、年間 300 万円上限)
中小機械装置: 7 年 (一般的) — 厳密には耐用年数省令
ソフトウェア: 5 年 (定額)
建物附属設備: 15 年 等
```

補助金との関連:
- **圧縮記帳** で課税繰延可能 (一定要件下) — 税理士相談

## E. 資金繰り表 (最重要)

### E-1. テンプレート

```
                    M+1   M+2   M+3   ... M+24
[前月繰越現預金]      A     B     C
─収入─
売上入金 (現金)
売上入金 (掛回収)
借入入金
補助金入金
その他
[収入合計]            X
─支出─
仕入支払 (現金)
仕入支払 (掛支払)
人件費
家賃
公共料金
支払利息
借入元金返済
税金 (法人税・消費税・源泉)
設備投資
その他
[支出合計]            Y
─差引─
当月収支              X-Y
[当月末残高]          A+X-Y = B
```

### E-2. 警戒ライン

| 状態 | アクション |
|---|---|
| 月末残高 < 1 ヶ月分の固定費 | **危険** — 即融資相談 |
| 月末残高 < 3 ヶ月分の固定費 | **要注意** — 増収 or 借入準備 |
| 月末残高 ≥ 6 ヶ月分の固定費 | 健全 |
| 月末残高 ≥ 12 ヶ月分の固定費 | 投資余力あり |

### E-3. ありがちな落とし穴

- **売上 ≠ 入金**: 売掛 60〜90 日後でようやく入金
- **消費税の納付**: 預かり消費税は手元にあるが期日には出ていく (益税減少)
- **法人税中間申告**: 8 ヶ月目に多額流出
- **賞与**: 6 月・12 月の固定費跳ね上がり
- **補助金の後払い**: 採択 → 実施 (立替) → 1 年後入金

## F. 借入返済シミュレーション

### F-1. 元利均等返済

```
毎月返済額 = 元金 × {月利 × (1+月利)^回数} ÷ {(1+月利)^回数 − 1}
```

毎月の返済額が一定。元金充当部分が徐々に増える。

### F-2. 元金均等返済

```
毎月元金返済 = 元金 ÷ 回数
毎月利息返済 = 残元金 × 月利
毎月返済額 = 元金返済 + 利息返済 (徐々に減少)
```

総支払利息は元利均等より少ない。

### F-3. 据置期間あり

```
据置期間中: 利息のみ支払
据置後   : 元利返済開始

例: 5,000 万円 / 金利 1.5% / 期間 10 年 / 据置 1 年
据置中: 月利息 約 6.25 万円
据置後: 元金 5,000 万を 9 年 (108 ヶ月) で返済
```

## G. 主要 KPI と業界平均

| KPI | 計算式 | 一般的な目安 |
|---|---|---|
| 粗利率 | (売上 − 売上原価) ÷ 売上 | 業種で大差: 製造 20-40%, サービス 40-70%, IT 50-80% |
| 営業利益率 | 営業利益 ÷ 売上 | 中小製造 3-7%, IT 10-20% |
| 自己資本比率 | 純資産 ÷ 総資産 | 30% 以上が望ましい |
| 流動比率 | 流動資産 ÷ 流動負債 | 200% 以上が安心 |
| 当座比率 | 当座資産 ÷ 流動負債 | 100% 以上 |
| ROA | 経常利益 ÷ 総資産 | 5% 以上で良好 |
| ROE | 当期純利益 ÷ 純資産 | 10% 以上で良好 |
| 借入金月商倍率 | 有利子負債 ÷ 月商 | 3〜6 ヶ月以内 |
| 労働生産性 | 付加価値 ÷ 従業員数 | 業種別比較 |
| 付加価値額 | 営業利益 + 人件費 + 減価償却 | 中小企業庁定義の場合 |

## H. 感度分析 (例)

```
変数        ベース  +10%   -10%   +20%   -20%
売上         100M   110M    90M   120M    80M
営業利益    5M     12M    -2M    19M    -9M  ← 売上 -10% で赤字転落
EBITDA      8M     15M     1M    22M    -6M

→ 売上 -10% でも凌げる費用構造を作るべき
```

## I. ストレスシナリオ

| シナリオ | 想定 | 対応策 |
|---|---|---|
| ベース | 計画通り | — |
| 楽観 | 売上 +30%, 想定費用 | 投資加速、借入繰上返済 |
| 悲観 | 売上 -30%, 大型解約 | 経費削減、追加借入 (公庫セーフティ) |
| 危機 | 売上 -50%, 主要顧客倒産 | 雇用調整助成金、リスケ交渉、廃業判断 |

各シナリオで 24 ヶ月先の月末残高を試算し、**現預金がマイナスにならない打ち手** を事前に決めておく。

## J. AI 補助のポイント

財務モデル作成における AI の使い方:

```
✅ 良い使い方
- セル数式の関数候補
- KPI 定義の説明
- シナリオ試算のロジック整理
- 業界別ベンチマーク値の取得 (要再検証)

❌ 悪い使い方
- 業績数値そのものの捏造
- 業界平均を「大体これくらい」で AI に答えさせ、根拠なく採用
- 顧客名・売上明細の入力 (機密)
- 計画書全文を AI 任せ
```

財務モデルは経営者本人が **数式を理解** していなければ、後の経営判断に使えない。AI 補助は「説明」と「整理」までに留め、**数値は自分の頭で決める**。
