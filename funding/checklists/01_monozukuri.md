# ものづくり・商業・サービス生産性向上促進補助金 (ものづくり補助金) チェックリスト

> ⚠️ 公募回ごとに要件・補助率・上限額が変動。最新公募要領を**必ず**確認すること。

## A. 申請可否の事前判定

```
□ 中小企業の定義 (業種別の資本金 OR 従業員数) に該当
   - 製造業: 資本金 3 億円以下 or 従業員 300 人以下
   - 卸売業: 1 億円以下 or 100 人以下
   - 小売業: 5 千万円以下 or 50 人以下
   - サービス業: 5 千万円以下 or 100 人以下
□ 設立 1 期 以上 (公募により異なる)
□ 過去採択からの待機期間 (連続採択制限) のクリア
□ 応募要件の「賃上げ」(事業実施期間内に給与支給総額 +X% / 最低賃金 +30 円) を遂行可能
□ 付加価値額 +3% / 年 等 の生産性目標を遂行可能
□ 採択率の目安 (近年 30〜60%) と費用対効果
```

## B. 申請準備 (締切から逆算 6〜8 週間前)

```
□ G-BizID プライム の取得 (発行に 2〜3 週間)
□ 公募要領 / 公募スケジュール の取得
□ 認定経営革新等支援機関 の選定 (要否確認)
□ 直近 3 期の決算書
□ 確定申告書一式 (法人税・消費税・所得税)
□ 履歴事項全部証明書 (登記)
□ 主要設備の見積書 (相見積 推奨)
□ 自己資金 / 借入計画
```

## C. 事業計画書 必須記載事項

```
□ 補助事業の具体的取組内容 (5W1H)
□ 革新性 (業界・地域の他社に対して新規)
□ 事業化に向けた具体的な見込み (誰にいくらで売る)
□ 政策面 (賃上げ・GX・DX・スタートアップ・地域貢献 等)
□ 付加価値額・労働生産性の向上計画 (3〜5 年)
□ 実施スケジュール (Gantt)
□ 体制 (誰が何を)
□ リスクと対応
```

## D. 必須経費区分

```
対象経費 (制度により幅あり、要確認):
□ 機械装置・システム構築費 (主要)
□ 技術導入費
□ 専門家経費
□ 運搬費
□ クラウドサービス利用費
□ 原材料費 (試作のみ)
□ 外注費 (試作製造のみ)
□ 知的財産権等関連経費

対象外:
□ 既存設備の単純更新
□ 不動産・建物
□ 自動車 (一部例外あり)
□ 補助対象期間外の発注・支払
□ 経費の二重申請 (他補助金との)
```

## E. 採択後 〜 完了報告

```
□ 採択通知 → 交付申請 (詳細経費の確定)
□ 交付決定通知 受領 → これ以降に発注
□ 中間検査 (要請があれば)
□ 補助対象期間内に完了 + 支払完了
□ 実績報告書 (期限厳守: 通常 30 日以内)
□ 補助金額 確定通知
□ 精算払い 受領
□ 事業化状況報告 (5 年間)
□ 関連書類 5 年保存 (税務 7 年と分離管理)
```

## F. 落とし穴 (経験則)

- **交付決定前の発注** = 全額対象外
- 見積書の **発行日が古い** (3 ヶ月以内推奨)
- 相見積取得義務 (一定額以上) のスキップ
- **賃上げ未達成 → 補助金一部返還**
- 海外発注の場合の為替変動・関税
- 付加価値額目標未達 → 是正報告 → 翌年以降の不利
