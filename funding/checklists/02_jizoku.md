# 小規模事業者持続化補助金 チェックリスト

> 商工会議所 / 商工会 経由の伴走型補助金。販路開拓・業務効率化に活用。

## A. 申請可否

```
□ 「小規模事業者」の定義 (業種別)
   - 商業・サービス業 (宿泊・娯楽除く): 常時使用する従業員 5 人以下
   - 宿泊業・娯楽業: 20 人以下
   - 製造業その他: 20 人以下
□ 商工会議所 (or 商工会) の管轄地域での事業
□ 直近の確定申告 / 決算 を完了
□ 法人税・所得税・消費税の滞納なし
```

## B. 申請ルート

```
□ 商工会議所 / 商工会 への会員登録 (非会員でも申請可だが、会員推奨)
□ 経営計画書の事前相談 (商工会経営指導員)
□ 「事業支援計画書 (様式4)」を商工会に発行依頼
□ 公募締切から逆算で 4 週間前に着手 (発行に時間)
□ 電子申請 (Jグランツ + GビズID)
```

## C. 必要書類

```
□ 経営計画書 (商工会の伴走で作成)
□ 補助事業計画書 (販路開拓の中身)
□ 事業支援計画書 (商工会発行)
□ 直近の確定申告書 / 決算書
□ 賃金引上げ枠の場合: 直近の賃金台帳等
□ 法人なら登記簿、個人なら開業届控
```

## D. 対象経費 (例: 一般枠)

```
対象:
□ 広報費 (チラシ・WEB 広告・看板)
□ ウェブサイト関連費
□ 展示会等出展費
□ 機械装置等費 (販路開拓に直結)
□ 開発費
□ 資料購入費
□ 雑役務費
□ 借料
□ 設備処分費
□ 委託・外注費

対象外:
□ 直接的な販売物
□ パソコン汎用品
□ 通常の事業運営費
□ 中古車・新車の購入
```

## E. 申請成功のポイント

```
□ 経営課題の明確化 → 改善策の論理性
□ 数値目標 (売上 / 客数 / 認知度) の設定
□ 商工会経営指導員の助言を反映
□ 計画書と見積書の整合
□ 写真・図表で具体性を出す
```

## F. 採択後

```
□ 交付決定通知 受領 → 発注開始
□ 補助事業期間内 (通常 6 ヶ月) で実施完了
□ 領収書・契約書・納品書 を保管
□ 実績報告書 + 経費明細 を期限内に
□ 入金 (確定通知後)
□ 1 年後・5 年後の事業効果報告
```
