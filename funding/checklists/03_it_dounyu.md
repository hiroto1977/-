# IT 導入補助金 チェックリスト

## A. 申請可否

```
□ 中小企業 / 小規模事業者 に該当
□ 連携する **IT 導入支援事業者 (登録ベンダ)** を選定済
   - 公式ポータルで事業者検索
   - 同事業者が **IT ツール** を登録済
□ 業種特有要件 (枠による) のクリア
□ SECURITY ACTION 自己宣言 (☆/☆☆) の実施
□ 「みらデジ」経営チェック の実施 (枠による)
□ 直近の決算が黒字 (枠により参考程度)
```

## B. 枠の選択 (一般的な分類例 — 公募で再編される)

```
□ 通常枠: 標準的な業務 IT 化
□ セキュリティ対策推進枠: 情報セキュリティ強化
□ インボイス対応枠: 電子インボイス対応の会計・受発注
□ 複数社連携 IT 導入枠: 複数事業者の共同導入
```

## C. 申請プロセス

```
□ G-BizID プライム 取得 (発行 2-3 週)
□ IT 導入支援事業者 と打合せ (ツール選定・要件すり合わせ)
□ 「みらデジ」経営チェック 受験 (枠により)
□ SECURITY ACTION 自己宣言
□ 申請マイページ作成 (支援事業者と二人三脚)
□ 事業計画 (生産性向上見込み・労働生産性) 入力
□ 必要書類 アップロード
□ 申請 → 採択 → 交付決定 → 契約 → 導入 → 報告
```

## D. 必要書類

```
□ 履歴事項全部証明書 (法人) / 開業届控 (個人)
□ 法人税納税証明書 (その1) / 所得税納税証明書
□ 直近の決算書
□ 賃金台帳 (賃上げ要件あり枠)
□ 業種特有書類
```

## E. 対象経費

```
対象:
□ ソフトウェア購入費・クラウド利用料 (1 年分)
□ 導入関連費 (設定・カスタマイズ・教育)
□ ハードウェア (枠による: PC・タブレット・レジ・券売機)
□ セキュリティ製品 (該当枠)

対象外:
□ 一般的な汎用ハードウェア (通常枠の場合)
□ 既存ソフトの保守更新
□ 中古品
□ 個人利用 (家庭用)
```

## F. 注意点

```
□ **支援事業者の選定が成否を左右** — 採択実績・サポート品質
□ 契約は **交付決定後** (前倒し NG)
□ ツール導入後 1〜数年は効果報告義務
□ 生産性向上目標 (労働生産性 +1% / 年 等) の達成義務
□ 不正受給 (虚偽の生産性報告 等) は厳罰
```
