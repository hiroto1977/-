# 事業承継・引継ぎ補助金 チェックリスト

## A. 主な枠 (例)

```
□ 経営革新枠 — 承継後の新規事業展開
□ 専門家活用枠 — M&A の DD・FA 報酬
□ 廃業・再チャレンジ枠 — 廃業費 + 新事業
```

## B. 申請可否

```
□ 中小企業 / 小規模事業者
□ 経営者交代 (親族内/親族外/M&A) を一定期間内に実施 (or 実施予定)
□ 承継完了後の事業継続意思
□ 認定経営革新等支援機関 の確認書
```

## C. 必要書類 (枠共通の例)

```
□ 事業計画書 (承継後の構想含む)
□ 直近の決算書 (承継元/承継先 双方)
□ 経営者交代の証跡 (株式譲渡契約書、登記簿、雇用変更 等)
□ 専門家経費の見積書 (専門家活用枠)
□ DD レポート (該当する場合)
```

## D. 対象経費

```
経営革新枠:
□ 設備費・原材料費・新規雇用人件費 等

専門家活用枠:
□ M&A 仲介手数料・FA 報酬
□ デューデリジェンス費用
□ 弁護士・税理士・公認会計士費用

廃業・再チャレンジ枠:
□ 廃業手続費 (在庫処分・原状回復)
□ 新事業準備費
```

## E. 注意点

```
□ M&A の場合、**最終契約 (DA) 前の DD 費用** が対象になる枠あり
   → 仲介開始の早期に補助金応募スケジュールを把握
□ 承継元・承継先 双方の同意・協力が必要
□ M&A 仲介業者 を **登録 M&A 支援機関** から選ぶことが要件の場合あり
□ 承継後 X 年以内の事業継続義務
```

## F. 関連スキーム

```
□ 公庫: 事業承継・集約・活性化支援資金
□ 民間: M&A ローン (仲介会社経由)
□ 経営者保証ガイドライン に基づく保証解除交渉 (承継時の論点)
□ 株式譲渡所得の税務 (税理士確認)
```
