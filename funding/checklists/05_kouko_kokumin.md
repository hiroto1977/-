# 日本政策金融公庫 国民生活事業 融資 チェックリスト

> 個人事業主・小規模事業者向け。創業融資・経営改善資金 等。

## A. 主要制度 (例)

```
□ 新創業融資制度 (創業期、無担保・無保証人 — 制度詳細は窓口確認)
□ 一般貸付 (運転・設備、業種ほぼ問わず)
□ 経営環境変化対応資金 (セーフティネット)
□ 小企業等経営改善資金 (マル経) — 商工会議所/商工会 推薦
□ 教育一般貸付 (個人)
□ 食品貸付・生活衛生貸付 (業種特化)
```

## B. 事前準備

```
□ 借入希望額の妥当性 (使途明確化)
□ 返済原資の試算 (営業利益 + 減価償却 で月次返済額を上回る)
□ 自己資金 (希望額の 1/3 〜 1/10 が目安)
   → 通帳で「コツコツ貯めた」証跡が好印象
□ 業界経験・職歴 (創業の場合)
□ 借入実績 (個人 / 法人 の信用情報をチェック)
```

## C. 申込書類

```
□ 借入申込書
□ 創業計画書 (創業の場合) or 企業概要書
□ 設備資金見積書 (設備資金の場合)
□ 履歴事項全部証明書 (法人) / 開業届控 (個人)
□ 直近 2 期の決算書 + 試算表 (既存事業者)
□ 法人税・所得税・消費税の納税証明
□ 店舗賃貸借契約書
□ 本人確認書類
□ 通帳のコピー (自己資金の出所)
```

## D. 面談 (担当者ヒアリング) の準備

```
□ 「なぜこの事業をするのか」の理由 (300 字で語れる)
□ 競合との違い・自社の強み
□ 売上計画の根拠 (誰に・いくらで・何個)
□ 失敗リスクの認識と対応策
□ 個人保証 / 担保についての方針 (経営者保証ガイドラインを学習)
□ 既存借入があれば、その状況と返済原資
□ 想定外シナリオ (売上半減 等) への対応
```

## E. 採択後

```
□ 契約書 (印紙税は借入額に応じて)
□ 入金確認
□ 約定返済 (口座振替、滞納厳禁)
□ 期日前返済 (一部 / 全部) のルール把握
□ 業績悪化時の **早期相談** (公庫はリスケに相対的に協力的)
□ 公庫担当者との関係維持 (年に 1〜2 回の面談)
```

## F. 落とし穴

```
□ 自己資金が 「直前に振り込み」 → タンス預金疑念で不利
□ 創業計画の数値根拠が薄い (「ITで年商 1 億」 等)
□ 同時期の複数借入申込 (信用情報で他行も照会される)
□ 返済猶予を申し出ずに延滞 → 信用情報悪化
```
