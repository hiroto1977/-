# 日本政策金融公庫 中小企業事業 融資 チェックリスト

> 中規模事業者向け。新事業活動・設備・環境エネルギー・資本性ローン 等。

## A. 主要制度 (例)

```
□ 新事業活動促進資金 — 新分野進出・第二創業・経営革新
□ 企業活力強化資金 — 設備投資・システム
□ 環境・エネルギー対策資金 — 省エネ・再エネ
□ 資本性ローン (挑戦支援資本強化特別貸付)
   → 純資産扱い・劣後債的性質、財務改善効果
□ 海外展開・事業再編資金 — 海外進出
□ ソーシャルビジネス支援資金
```

## B. 国民生活事業との違い

| 項目 | 国民生活事業 | 中小企業事業 |
|---|---|---|
| 主対象 | 個人・小規模 | 中堅含む |
| 1 件 上限 | 数千万円 〜 1 億円程度 | 数億円規模も |
| 担保 | 多くは無担保 / 軽担保 | 通常は担保 + 経営者保証 |
| 期間 | 短〜中期 | 中〜長期 |
| 審査 | 個人の信用 + 簡易事業計画 | 詳細な事業計画 + 財務 |

## C. 申込書類

```
□ 借入申込書
□ 詳細事業計画書 (新事業の場合は手厚く)
□ 直近 3 期の決算書 + 試算表
□ 法人税納税証明書 (その1)
□ 投資計画書・見積書
□ 担保・保証人 関連書類
□ 設備の現地写真 / 図面 (設備資金)
□ 海外案件は 進出先国情報・現地法人計画
```

## D. 資本性ローン (劣後ローン) の特徴

```
□ 期日一括返済 (元金返済期日まで利息のみ)
□ 期間: 5〜20 年 等 (長期)
□ 純資産扱い (BS 上)
□ 借入と表記されないため、自己資本比率改善
□ 金利は通常融資より高め (業績連動)
□ 主に成長投資・事業再生・スタートアップ向け
```

## E. 中小企業事業 申込フロー

```
□ 公庫支店 と事前相談 (大型案件は本店審査)
□ 案件サマリ (1〜2 ページ) を持参
□ 担当者・支店長との面談 (複数回)
□ 内部審査 (数週〜数ヶ月)
□ 融資条件提示 → 交渉
□ 契約 → 入金
□ モニタリング (四半期 / 半期 ヒアリング)
```

## F. 審査でよく見られる項目

```
□ 業界における自社の位置づけ (シェア・差別化)
□ 経営者の経歴・資質
□ 既存取引銀行 (協調 / 補完)
□ 設備投資の必要性・回収可能性
□ ストレスシナリオ (売上 -20% でも返済可能か)
□ 経営者保証ガイドライン適用の可否
```

## G. 経営者保証ガイドライン

```
□ 法人と経営者の資産・経理の明確な分離
□ 法人と経営者の間の必要十分な資本余裕
□ 適時適切な情報開示
   → 上記を満たせば「経営者保証なし」を交渉可能
```

詳細: 全国銀行協会・日本商工会議所 共同策定の「経営者保証に関するガイドライン」を参照。
