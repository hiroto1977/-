# 民間金融機関 融資 チェックリスト

> メガバンク / 地方銀行 / 信用金庫 / 信用組合 / 保証協会付制度融資 共通。

## A. 金融機関の選び方

```
□ 既存取引のあるメインバンク (関係性 重要)
□ 地域密着の信用金庫・信用組合 (小規模に強い)
□ 自治体の制度融資 (利子補給・保証料補助)
□ 政府系の補完: 公庫と組み合わせ
□ 近接する複数行と関係維持 (1 行依存リスク回避)
```

## B. プロパー融資 vs 保証協会付

| | プロパー | 保証協会付 |
|---|---|---|
| 審査 | 銀行が単独 | 銀行 + 保証協会 (二段階) |
| 担保 | 銀行判断 | 保証協会の保証 |
| 金利 | 銀行裁量 | + 保証料 (年 0.5〜2%) |
| 借入の上限 | 大きい | 制度・保証協会の限度 |
| 創業期 | 困難 | 利用しやすい |

## C. 信用保証協会付の流れ

```
□ 自治体窓口 or 商工会 で「斡旋書」取得 (制度融資の場合)
□ 金融機関 申込
□ 金融機関 → 保証協会 へ保証申込
□ 保証協会 審査 (信用情報・事業内容)
□ 保証承諾 → 金融機関 融資実行
□ 利用者は 金融機関 へ返済
□ 万一 返済不能 → 保証協会 が代位弁済 → 利用者は 保証協会 へ返済継続
```

## D. 申込書類

```
□ 法人:
   - 履歴事項全部証明書
   - 直近 2〜3 期 決算書 + 勘定科目内訳書
   - 試算表 (直近 + 申込月)
   - 法人税納税証明書 (その1, 3, 3-2 等)
   - 商業登記簿謄本
□ 個人事業:
   - 開業届控
   - 直近 2 期 確定申告書
   - 所得税納税証明書
□ 共通:
   - 借入申込書
   - 資金使途明細 (設備見積書 等)
   - 事業計画書
   - 既存借入一覧
   - 預金通帳
□ 設備案件:
   - 見積書 (相見積)
   - 設備写真 / 図面 / カタログ
□ 担保案件:
   - 不動産登記簿 / 評価額資料
```

## E. 面談での想定質問

```
□ なぜ今、この金額が必要か
□ 資金使途の合理性
□ 返済原資 (営業利益 + 減価償却 vs 月次返済額)
□ 他行借入の状況・順位 (担保順位)
□ 売上減少時の対応 (24 ヶ月の資金繰り)
□ 経営者個人の資産・負債 (個人保証関連)
□ 相続・後継者の見通し (長期借入)
□ 業界の見通し
□ 顧客集中リスク (上位 3 社で売上の何 %)
```

## F. 借入条件で確認すべき事項

```
□ 金利 (固定 / 変動 / TIBOR スプレッド)
□ 返済方法 (元利均等 / 元金均等 / 期日一括)
□ 元金据置期間
□ 保証料 (協会付の場合)
□ 印紙代 (借入額に応じる)
□ 期限の利益喪失事由
□ コベナンツ (財務制限条項)
   - 純資産維持
   - 配当制限
   - 設備投資制限
□ 担保設定 (順位・登記費用)
□ 連帯保証人の範囲
□ 期日前返済の手数料・違約金
```

## G. 良好な関係維持の習慣

```
□ 月次・四半期試算表の自主提供
□ 業績悪化兆候の早期共有
□ 大きな投資・新規取引の事前相談
□ 担当者の異動把握 (引継ぎが命)
□ 預金平残の維持 (借入額の 1〜2 ヶ月分)
□ 決算後 1 ヶ月以内の決算報告 (経営者本人で)
```

## H. やってはいけないこと

```
□ 同時期に複数行へ同じ案件を出す (信用情報で他行も知る)
□ 決算書の数値を行ごとに変える (発覚で信用ゼロ)
□ 試算表を出さない・遅らせる
□ 業績悪化を隠す
□ 不渡 (1 度で取引困難、2 度で銀行取引停止)
□ 借入金を別の借入返済に流用 (使途違反)
```

## I. リスケジュール (返済猶予)

```
□ 兆候: 月次返済の負担が重い、資金繰り悪化
□ 早期 (3 ヶ月前) に銀行に相談
□ 経営改善計画書 (1〜3 年) を提出
□ 中小企業活性化協議会 への相談
□ メイン行 + 取引行 全体での合意形成
□ 公庫の経営改善資金の併用
```

延滞してから相談 = 信用情報悪化。**支払えなくなる「前」に動く**。
