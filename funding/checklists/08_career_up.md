# キャリアアップ助成金 チェックリスト

> 厚労省。非正規雇用労働者のキャリアアップ支援。社労士同伴推奨。

## A. 主なコース (再編・廃止あり、最新を要確認)

```
□ 正社員化コース — 有期 → 無期 / 有期 → 正社員 / 無期 → 正社員
□ 賃金規定改定コース — 賃金テーブルの全体改定
□ 賃金規定共通化コース — 正社員と非正規の同一化
□ 賞与・退職金制度導入コース
□ 短時間労働者労働時間延長コース
□ 社会保険適用時処遇改善コース
```

## B. 共通の事前要件

```
□ 雇用保険適用事業所
□ 「キャリアアップ計画書」を計画期間開始前に管轄労働局へ提出
□ キャリアアップ管理者の選任 (社内)
□ 就業規則 (10 人以上事業所) の労働基準監督署届出済
□ 直近 6 ヶ月間に**事業主都合の解雇** がない
□ 過去の不正受給歴がない
```

## C. 正社員化コース 個別要件 (例)

```
□ 対象者: 6 ヶ月以上雇用された有期 / 無期雇用労働者
□ 転換前の試用期間ではない
□ 転換前後で同一の労働者
□ 就業規則 / 労働協約に転換規定あり
□ 転換後 6 ヶ月の継続雇用 + 賃金支払
□ 転換後の賃金が転換前より 3% 以上 増加 (要件)
□ 中小企業の場合の加算
```

## D. 申請プロセス (正社員化コースの例)

```
□ 1. キャリアアップ計画書 提出 (転換前)
□ 2. 就業規則改定 (転換規定の整備)
□ 3. 対象労働者を 転換 (or 直接雇用)
□ 4. 転換後 6 ヶ月の雇用継続 + 賃金支払
□ 5. 6 ヶ月経過後 2 ヶ月以内 に支給申請書 提出
□ 6. 労働局 審査
□ 7. 支給決定 → 入金 (申請から 6 ヶ月程度)
```

## E. 必要書類 (例)

```
□ 支給申請書 (様式)
□ キャリアアップ計画書 (写)
□ 就業規則 (転換規定明示)
□ 転換前後の労働条件通知書 / 雇用契約書
□ 賃金台帳 (転換前 6 ヶ月 + 転換後 6 ヶ月)
□ 出勤簿 / タイムカード
□ 雇用保険被保険者資格取得・喪失確認通知書
□ 中小企業確認書類
```

## F. 落とし穴

```
□ キャリアアップ計画書 未提出 で転換 → 不支給
□ 就業規則 に転換規定なし → 不支給
□ 転換後の賃金 3% 未達 → 不支給
□ 申請期限 (転換後 6 ヶ月後の応当日翌日から 2 ヶ月) 徒過 → 失権
□ 転換後 6 ヶ月以内の解雇 / 自己都合退職 → 不支給
□ 同一労働者の重複申請 (他コース)
□ 不正受給 → 返還 + 加算金 + 5 年間 不支給 + 公表
```

## G. 社労士活用

```
□ 就業規則改定 + 計画書作成 + 申請まで一括依頼
□ 報酬: 着手金 + 受給額の 10〜20% が目安
□ 社内整備が並行で進む副次効果
□ 制度変更 (年度・年度途中) のキャッチアップ
```
