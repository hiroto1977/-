# 雇用調整助成金 チェックリスト

> 経済上の理由により事業活動の縮小を余儀なくされた事業主が労働者を休業等させる場合の助成。

## A. 受給要件 (通常時)

```
□ 雇用保険適用事業所
□ 直近 3 ヶ月の生産量・販売量等が前年同期比 10% 以上 減少
□ 雇用保険被保険者数 + 一定の派遣労働者 が前年同期比で 一定比率を超えて 増加していない
□ 過去に雇用調整助成金 を受けた場合、一定のクーリング期間
□ 6 ヶ月以上雇用された労働者 (一部例外)
```

> ⚠️ 災害・パンデミック等の特例期間は要件大幅緩和あり。

## B. 対象となる雇用調整 (休業等)

```
□ 休業 (1 日 / 部分)
□ 教育訓練 (休業の代わりに業務外研修)
□ 出向 (3 ヶ月以上 1 年以内、3 者間契約)
```

## C. 申請プロセス

```
□ 1. 休業計画届 / 教育訓練計画届 / 出向計画届 を 事前 に労働局へ提出
□ 2. 計画期間中に 休業等 を実施
□ 3. 休業手当 を支払 (平均賃金の 60% 以上)
□ 4. 判定基礎期間 (賃金締切期間) ごとに支給申請
□ 5. 労働局 審査
□ 6. 支給決定 → 入金
```

## D. 助成額

```
助成額 = 休業手当相当額 × 助成率
助成率: 中小 2/3, 大企業 1/2 (通常時)
1 人 1 日あたり 上限あり (年度・特例で変動)
教育訓練の場合: 加算額あり
```

## E. 必要書類 (例)

```
□ 計画届 (各様式)
□ 支給申請書
□ 休業協定書 (労使協定)
□ 休業実績一覧 (誰がいつ休業)
□ 賃金台帳 / 出勤簿 (休業手当支払証跡)
□ 売上等の生産指標 確認資料 (前年同期比減少の証拠)
□ 雇用保険被保険者資格関連書類
□ 中小企業の確認資料
```

## F. 落とし穴

```
□ **計画届 を事前 に出さず実施** → 不支給
□ 休業手当未払 (60% 未満) → 不支給
□ 解雇 (申請対象期間中は禁止) → 不支給 + 過去分 返還
□ 売上減少の証拠 (生産指標) が不十分
□ 休業日に労働者を別作業に従事させる (勤怠捏造) → 不正受給
□ 役員のみの休業 → 対象外 (雇用保険被保険者でないため)
□ 申請期限 (判定基礎期間末日翌日から 2 ヶ月以内 等) 徒過
```

## G. 関連制度

```
□ 産業雇用安定助成金 (出向元・出向先 双方支援)
□ 雇用維持と セーフティネット融資 を組合わせ
□ 経営改善計画 を並行策定
```

## H. 平時からの備え

```
□ 就業規則 に休業手当規定
□ 労使協定 ひな形
□ 売上推移の月次データを 36 ヶ月以上保管
□ 顧問社労士との緊急連絡体制
```
