# 経営者保証 解除 / 不徴求 申請書 テンプレ

> 「経営者保証に関するガイドライン」(全国銀行協会・日本商工会議所) に基づく、
> 経営者保証の解除 (新規借入時の不徴求 / 既存借入の解除) を金融機関に依頼する文書のひな形。

---

## A. 経営者保証ガイドラインの 3 要件

```
1. 法人と経営者個人の資産・経理が明確に分離されている
2. 法人と経営者個人の間の必要十分な資本余裕がある
3. 法人から適時適切な情報開示が行われている
```

これらを満たしていることを証拠と共に示す。

---

## B. 申請書 ひな形

```
[YYYY-MM-DD]

[銀行名] [支店名]
支店長 [氏名] 様
担当 [氏名] 様

                                   [社名]
                                   代表取締役 [氏名] 印

         経営者保証に関する取扱いの見直しのお願い

平素より格別のご高配を賜り、厚く御礼申し上げます。

このたび、当社 [今回の借入案件名 or 既存借入のご相談] に関連し、
「経営者保証に関するガイドライン」に基づく経営者保証の
[ ] 不徴求 (新規借入時)
[ ] 解除 (既存借入)
をご検討いただきたく、下記のとおり当社の状況をご報告申し上げます。

ご審議のほど、どうぞよろしくお願い申し上げます。

                  記

1. 法人と経営者個人の資産・経理の明確な分離
   (1) 経営者貸付・借入: なし (or 解消済)
   (2) 不動産の私物利用: なし
   (3) 個人費用の法人計上: なし (顧問税理士確認済)
   (4) 法人預金口座と個人口座の明確な分離

2. 法人と経営者個人の間の必要十分な資本余裕
   (1) 自己資本比率: ___ % (基準 30% 以上を満たす場合明示)
   (2) 純資産額: ___ 円
   (3) 借入金月商倍率: ___ 倍 (基準 6 倍以下が望ましい)
   (4) 流動比率: ___ %

3. 法人からの適時適切な情報開示
   (1) 月次試算表を翌月 ___ 日までにご提出
   (2) 決算書を期末後 ___ ヶ月以内にご提出
   (3) 重要な投資・契約の事前ご相談
   (4) 業績悪化時の早期ご報告

4. その他参考情報
   (1) 直近 3 期 連続黒字 (該当する場合)
   (2) 個人保証の代替手段の検討状況
       例: 流動資産担保融資、ABL、根抵当権設定 等

5. 添付資料
   ・直近 3 期 決算書
   ・直近月次試算表
   ・財務指標まとめ表
   ・税理士からの所見書 (取得可能な場合)
                                                以上
```

---

## C. 申請の事前準備

```
□ 顧問税理士に「3 要件への適合度」をレビュー依頼
□ 中小企業診断士・公認会計士のセカンドオピニオン
□ 主要金融機関 (メイン以外も) と方針すり合わせ
□ 過去 3 期の財務データ整理
□ 経営者個人と法人の資金移動の総点検
□ 個人費用の混入があれば事前是正 (修正申告 等)
```

## D. 認められやすい状況

| 状況 | 評価 |
|---|---|
| 黒字基調 + 自己資本比率 30%+ + 月商倍率 6 倍以下 | 良 |
| 法人個人の分離が会計帳簿で明確 | 良 |
| 月次の数字を毎月銀行へ自主提出 | 良 |
| 顧問税理士の所見書あり | 良 |
| 経営者の個人資産が手厚い (代替担保提供可) | 良 |

## E. 認められにくい状況 (改善余地)

```
□ 経営者貸付・経営者借入の残高 → 解消
□ 個人費用の法人計上が散見 → 修正
□ 業績悪化トレンド → 改善計画提出
□ 銀行への情報開示不足 → 月次共有を始める
□ 個人保証なしで他行貸出実績がない → 並行交渉
```

## F. 代替手段

経営者保証の代わりに以下を提案できる場合:

```
□ 動産担保融資 (ABL) — 在庫・売掛金担保
□ 不動産担保 (個人 or 法人)
□ 流動性担保 (定期預金 等)
□ 信用保証協会の保証 (個人保証なしの新制度)
□ 経営者個人の連帯保証ではなく 第三者の連帯保証
□ 借入額の縮小
```

## G. 解除されない場合の対応

```
□ 別の金融機関で打診 (主取引先は変えず、補完取引)
□ 公庫の経営者保証に関する特例 (ガイドライン適合企業向け)
□ 信用保証協会の保証 (経営者保証なし枠)
□ 段階的削減 (まず一部のみ解除)
```

---

## H. 注意事項

- **経営者保証ガイドラインは「お願いベース」**。法的強制力はなく、最終判断は金融機関
- 一度解除されても、業績悪化等で再徴求の可能性
- 解除された後も **誠実な情報開示と分離** を継続することが信頼維持の鍵
- 専門家 (中小企業診断士・公認会計士・弁護士) の関与を推奨

詳細・最新動向: 中小企業庁、日本商工会議所、全国銀行協会の公式情報を必ず参照。
