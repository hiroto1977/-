# 汎用業務テンプレ集

業務で繰り返し使う「型」を集約。コピー → 自社用にカスタマイズして使う。

## 構成

| カテゴリ | 内容 |
|---|---|
| [`hr/`](hr/) | 人事 — 求人票・面接シート・雇用契約・1on1・退職面談 |
| [`legal/`](legal/) | 法務 — NDA・業務委託契約・プライバシーポリシー・利用規約・特商法表記 |
| [`ops/`](ops/) | オペレーション — SOP・月次レビュー・KPI・インシデント・改善提案 |
| [`sales/`](sales/) | 営業 — 提案書・見積書・請求書・顧客アンケート |
| [`crisis/`](crisis/) | 危機対応 — 売上急落・取引先倒産・主要顧客喪失・SNS 炎上 |
| [`marketing/`](marketing/) | マーケ — プレスリリース・LP 構造・メルマガ・SNS |
| [`finance/`](finance/) | 財務 — 24 ヶ月資金繰り CSV・月次 PL ひな形 |

## 利用方針

- 機密情報を含む内容を AI に入れる場合は **ローカル LLM (Ollama) 経由**
- 法的拘束力のある書面 (契約・規約) は **必ず弁護士確認**
- 業界別ローカルルールがある場合はそれを優先
