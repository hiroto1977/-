# 取引先倒産時の対応手順

## A. 倒産情報の入手 (報道・噂・取引中断)

```
□ 事実確認 (信用調査会社 / 取引先公式 / 帝国データバンク 等)
□ 倒産形態の特定 (任意整理 / 民事再生 / 会社更生 / 破産)
□ 自社の債権額を即時集計
□ 担保 / 保証 の有無確認
```

## B. 即日対応 (24 時間以内)

```
□ 取引停止の判断 (新規発注・出荷の停止)
□ 売掛金 (未回収) の保全策
   - 担保処分の準備
   - 保証人への請求
   - 取引先の在庫・備品への動産先取特権
   - 動産売買先取特権の主張
□ 当社が支払うべき買掛金 (取引先 → 自社) の支払猶予検討
   ※ 倒産後の支払いは破産管財人を通すことになる
□ 法務 / 弁護士への相談
```

## C. 1 週間以内

### C-1. 債権届出
```
□ 倒産形態に応じた届出
   - 破産: 破産債権届出 (期限あり)
   - 民事再生: 再生債権届出 (期限あり)
   - 会社更生: 更生債権届出 (期限あり)
□ 添付書類: 契約書 / 請求書 / 納品書 / メール
□ 弁護士への代理依頼推奨
```

### C-2. 自社への影響度評価
```
□ 当該取引先の売上比率
□ 代替取引先の有無
□ 在庫処分・廃棄の可能性
□ 自社のキャッシュフロー影響
```

## D. 1 ヶ月以内

### D-1. 損失計上
```
□ 貸倒損失の計上 (税理士確認)
   - 法律上の貸倒
   - 事実上の貸倒
   - 形式上の貸倒
□ 貸倒引当金の設定
□ 期中の損益への影響反映
```

### D-2. 連鎖倒産の予防
```
□ 自社の資金繰り再シミュレーション
□ 主要取引銀行への状況共有 + 必要なら追加融資相談
□ 公庫: 経営環境変化対応資金 (セーフティネット)
□ 信用保証協会: セーフティネット保証 (取引先倒産特別保証)
□ 中小企業倒産防止共済 (経営セーフティ共済) 加入済なら共済貸付請求
```

### D-3. 顧客 / 取引先への説明
```
□ 当該取引先経由の顧客への状況説明
□ 代替供給体制の周知
□ 競合他社への顧客流出防止
```

## E. 3 ヶ月以内

```
□ 取引先選定基準の見直し (信用調査の頻度・基準)
□ 与信管理の強化 (1 社集中の解消)
□ 同業他社との情報共有 (業界団体経由)
□ 経営セーフティ共済 (未加入なら検討)
□ 取引基本契約への期限の利益喪失条項の追加
```

## F. 経営セーフティ共済 (中小機構)

- 月額掛金: 5,000 円 〜 20 万円
- 取引先倒産時に **掛金の最高 10 倍 (上限 8,000 万円)** を貸付
- 掛金は損金算入可能 (節税効果)
- 平時からの加入を強く推奨

## G. 弁護士相談ポイント

```
□ 動産先取特権の主張可否
□ 相殺可能性
□ 保証人への請求順序
□ 詐害行為取消権 (取引先が直前に資産を散逸させていた場合)
□ 否認権 (破産直前の取引が無効になる可能性)
```
