# 売上急落時の経営判断フロー

> 「先月対比 -30% 以上」または「3 ヶ月連続前年比 -20% 以上」を急落と定義。

## A. 即日 (発覚 24 時間以内)

```
□ 経営層緊急会議 (代表 + 財務 + 主要部門長)
□ 急落の事実確認 (一過性 / 構造的 / 統計エラー の切り分け)
□ 主要顧客への状況確認 (取引先倒産・競合変化・自社品質問題)
□ 主要金融機関への第一報 (隠蔽より先手)
□ 月次資金繰りの再試算 (24 ヶ月)
□ 緊急に必要な現金額の見積
```

## B. 1 週間以内

```
□ 原因の構造的分析 (PEST + 5 forces で外部、SWOT で内部)
□ 顧客 / 製品別の影響度マトリクス作成
□ コスト削減候補の洗い出し (固定費 / 変動費)
□ 雇用調整助成金 等のセーフティネットの初期検討
□ 公庫 / 信用保証協会 セーフティネット融資 の事前相談
□ 顧問税理士・社労士・弁護士・診断士 緊急ミーティング
```

## C. 2〜4 週間で実施

### C-1. 経費削減の優先順位
```
1. 投資的支出の凍結 (新規設備・採用・広告)
2. 任意経費の削減 (会議費・交際費・出張)
3. 役員報酬の自主減額 (税務リスク確認)
4. オフィス縮小・賃料交渉
5. 早期退職・希望退職 (最終手段)
   ※ 整理解雇 4 要件を満たすかは弁護士確認必須
```

### C-2. 売上回復策
```
□ 既存顧客への再販 / アップセル
□ 新規顧客開拓のリスト見直し
□ 商品 / サービスのピボット
□ 副業・新規事業の早期マネタイズ
```

### C-3. 資金調達
```
□ 公庫セーフティネット貸付
□ 信用保証協会のセーフティネット保証 (4 号 / 5 号 等)
□ メイン行の追加融資
□ ファクタリング (売掛金売却) — 最終手段
□ オーナー自身からの資本注入 (税務確認)
□ 補助金: 持続化補助金 等
```

## D. 1〜3 ヶ月で実施

```
□ 中期経営計画の修正
□ ビジネスモデルの見直し
□ 経営改善計画書の策定 (リスケ前提なら金融機関と)
□ 中小企業活性化協議会 への相談
□ 株主・取締役会への報告
```

## E. 最後の選択肢 (覚悟の判断)

```
□ 私的整理 (中小企業活性化協議会・REVIC)
□ 民事再生 (法的整理)
□ 自己破産
```

→ 早期に **弁護士・税理士・公認会計士** と相談。家族・社員・取引先への影響大。

## F. 相談先 まとめ

| 相談先 | 何を相談するか |
|---|---|
| 顧問税理士 | 数字の整理・税務上の救済策 |
| 顧問社労士 | 雇用調整・整理解雇の手続 |
| 顧問弁護士 | 契約・法的整理 |
| 中小企業診断士 | 経営改善計画 |
| 公庫 | セーフティネット融資・リスケ |
| 信用保証協会 | セーフティネット保証 |
| 商工会議所・よろず支援拠点 | 公的相談 (無料) |
| 中小企業活性化協議会 | 私的整理・再生支援 |
| 取引銀行 | 一斉に共有 |

## G. 心構え (経営者向け)

- **隠さず・早く・誠実に** ── 隠蔽は致命傷
- **経営者の心身を守る** ── 自殺・うつのリスク。SOS を出す
- **撤退の判断は早い方が傷が浅い**
- **再起の道は必ずある** ── 早期相談で再チャレンジ可能性も増える

参考: 「中小企業の事業再生等に関するガイドライン」(2022 年策定)
