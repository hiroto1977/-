# 24 ヶ月 資金繰り表 テンプレ

> Excel / Google Sheets で作成。本書は構造とロジックのみ。

## A. 行構造

```
A 列: 項目
B 列: 区分 (収入 / 支出)
C 列〜AC 列: M+1 〜 M+24 (月次)
AD 列: 24 ヶ月合計
AE 列: 12 ヶ月平均
```

## B. 行の例

```
■ 前月繰越現預金                    (各列 = 前月の月末残高)

【 収 入 】
売上入金 (現金)
売上入金 (掛 1 ヶ月後)              (前月売上 × 掛比率)
売上入金 (掛 2 ヶ月後)
借入入金
補助金入金 (申請中の場合は採択前後)
助成金入金
その他収入 (利息・配当・固定資産売却)
─────────
収入合計

【 支 出 】
仕入支払 (現金)
仕入支払 (掛 1 ヶ月後)
人件費 (給与・賞与)
法定福利費 (健保・厚年・労保)
家賃・地代
水道光熱費・通信費
広告宣伝費
販売手数料・カード手数料
研究開発費
旅費交通費
減価償却費 ※キャッシュフローには影響しないので除外
支払利息
借入元金返済
税金 (法人税・消費税・住民税・源泉)
設備投資
リース料
その他経費
─────────
支出合計

【 差 引 】
当月収支               (収入合計 - 支出合計)

■ 当月末現預金残高     (前月繰越 + 当月収支)
```

## C. キャッシュ警戒ラインの計算

```
固定費月額 = 人件費 + 家賃 + 水道光熱 + 通信 + 保険料 + 借入返済 + 等

警戒ライン:
  Red    = 固定費 × 1 ヶ月分
  Yellow = 固定費 × 3 ヶ月分
  Green  = 固定費 × 6 ヶ月分

各月末残高がどのラインかをセル背景色で可視化
```

## D. 仮定 (シート別管理)

```
売上前提:
- 月成長率 [ ]%
- 季節変動指数 (1 月 0.8 / 3 月 1.2 等)
- 主要顧客集中度 (上位 1 社で売上 X%)

回収条件:
- 現金: [ ]%
- 掛 1 ヶ月: [ ]%
- 掛 2 ヶ月: [ ]%

支払条件:
- 仕入現金: [ ]%
- 仕入掛 1 ヶ月: [ ]%

借入:
- 残高 [ ] 円 / 金利 [ ]% / 期間 [ ] 年 / 据置 [ ] ヶ月

人件費:
- 給与単価上昇 年 [ ]%
- 賞与 年 [ ] 回 / [ ] ヶ月分
```

## E. シナリオ別シートを 3 つ用意

```
1. ベースケース (現状の延長)
2. 楽観 (売上 +20% / 顧客拡大)
3. 悲観 (売上 -20% / 主要顧客喪失)

各シナリオでキャッシュ警戒ラインを下回る月を特定 → 事前打ち手の検討
```

## F. 月次 アップデート ルーチン

```
□ 月初 5 日以内に前月実績を入力
□ 来月以降の予測を見直し
□ 警戒ライン抵触月の打ち手を経営会議で決定
□ 主要金融機関へ自主提供 (関係構築)
```

## G. AI 補助のポイント

- 仮定パラメータの設定 (業界平均からの推定)
- 異常値の検出 (前年同月比 ±N%)
- 文章での解釈レポート

→ 個別具体の数値は経営者本人が決める。AI は構造と説明補助。
