# 特定商取引法に基づく表記 ひな形

> 通信販売 (ネット販売) を行う事業者は表示義務あり (特商法 11 条)。

```
            特定商取引法に基づく表記

【販売事業者】
社名 / 屋号: [ ]
代表者氏名: [ ]
所在地: [ ]
(個人事業主の場合、自宅住所の表示が原則。請求があった場合に
遅滞なく書面または電子で開示できる体制があれば、サイト上は
都道府県等の表示でも可)

【連絡先】
電話番号: [ ]
   (請求があったとき遅滞なく書面または電子で開示できる体制を
    取れば、サイト上は省略可能)
メール: [ ]
受付時間: [ ]

【販売価格】
各商品ページに記載
消費税: [ ] (内税 / 外税)

【商品代金以外の必要料金】
送料: [円] (詳細は別途記載)
代引手数料: [円]
消費税

【支払方法】
[クレジットカード / 銀行振込 / コンビニ / 代引 / その他]

【支払時期】
[商品注文時 / 商品到着時 / 月末締め翌月払い 等]

【引渡し時期】

【返品・交換】
[返品可能期間・条件]

お客様都合: [可能 / 不可]、可能なら期間と送料負担を明記。

【返品・交換の連絡先】
[ ]

【動作環境 (デジタル製品の場合)】
[ ]

【サブスクリプションの場合の追加事項】
契約期間: [ ]
自動更新: [あり / なし]、解約方法: [ ]
中途解約: [可否] / 違約金: [ ]
```

## 注意点
- 個人事業主の自宅住所表示の代替対応 (プラットフォームによる規定もある)
- 通信販売は **クーリングオフが原則ない** ためトラブル防止のため返品ポリシー明記
- BtoC は景品表示法・薬機法・健康増進法 等の表示規制も
