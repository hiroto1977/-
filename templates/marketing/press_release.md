## チェックリスト
```
□ 見出しが 40 字以内 + 数字あり
□ リードが 5W1H 全て含む
□ 専門用語に注釈
□ 引用ソースの明記
□ 取材問合せの窓口明記
□ 配信前の関係者最終確認 (代表 + 法務 + 関係取引先)
□ 配信予定先メディア リスト
□ 配信時刻 (火曜午前が伝統的に開封率高め)
```

## 配信チャネル
- 自社サイト ニュース欄
- SNS 公式アカウント
- プレスリリース配信サービス (PR TIMES / @Press / Value Press 等)
- 業界専門メディア 個別配信
- 既存記者へのメール
