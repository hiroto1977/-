# 月次経営レビュー 議事録 ひな形

```
月次経営レビュー [YYYY 年 M 月度]
日時: [YYYY-MM-DD HH:MM 〜 HH:MM]
場所: [本社会議室 / オンライン]
出席者: [氏名 / 役職]
書記: [ ]

== 1. 数値レビュー (15 分) ==
売上 (実績 / 計画 / 前年同月)
営業利益 (実績 / 計画 / 前年同月)
キャッシュ残高 (月末)
主要 KPI 3〜5 つ

数値分析:
- 計画比 +X% / -X%
- 主要要因 ([ ])
- 来月の見通し ([ ])

== 2. 案件・顧客 (15 分) ==
新規受注: [ ]
失注: [ ] / 理由 [ ]
クレーム / リスク案件: [ ]
重要顧客の状況: [ ]

== 3. 人 (10 分) ==
入退社・異動: [ ]
注目すべきパフォーマンス: [ ]

== 4. オペレーション (10 分) ==

インシデント発生: [ ]

== 5. 財務・資金繰り (10 分) ==
月末残高 vs 警戒ライン
24 ヶ月 資金繰り見通し
借入返済予定
申請中の補助金 / 助成金 / 融資

== 6. 戦略の進捗 (10 分) ==
中期計画の達成度
今月の打ち手と効果
来月の重点施策

== 7. リスク・課題 ==
新規発見リスク: [ ]

== 8. 決定事項 ==

|---|---|---|
| | | |

== 9. アクションアイテム ==
| 内容 | 担当 | 期限 |
|---|---|---|
| | | |

次回: [YYYY-MM-DD]
```

## 運用ルール
- 議事録は **会議終了後 24 時間以内** に出席者へ配布
- 数値はダッシュボードで自動生成して持参
- 過去 3 ヶ月分のアクションアイテム持ち越し率 を毎回確認
- 機密度: C2 〜 C3 (個人実績含むなら C3)
